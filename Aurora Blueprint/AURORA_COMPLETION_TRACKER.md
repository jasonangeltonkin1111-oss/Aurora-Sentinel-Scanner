# AURORA COMPLETION TRACKER

## PURPOSE

This file is the permanent handover-safe progress ledger for the Aurora Blueprint project.

It must be updated at the end of every meaningful Aurora Blueprint work run.

Its purpose is to let a new chat continue the project without relying on bloated chat history.

This file records:
- what has already been created
- what has already been extracted
- what is still missing
- what the current priority is
- what the next chat should do next
- what level of completion exists by module, wave, and archive area

This file is not theory.
This file is not a diary.
This file is not a vague summary.
It is the operational continuity record.

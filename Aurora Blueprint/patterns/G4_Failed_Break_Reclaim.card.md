# PATTERN NAME
Failed Break / Reclaim

# STATUS
APPROVED_GROUP

# PARENT FAMILY / FAMILIES
- Failed Break / Trap Reversal

# CANONICAL THESIS
This pattern group captures local structures where an attempted directional break cannot sustain itself, loses acceptance, and is reclaimed back through the initiating area, shifting the pattern meaning toward reversal or return-to-structure logic.

# PRIMARY MARKET HABITAT
- Failed Break / Trap State
- false release reverting through initiating structure

# PRIMARY STATE / SURFACE DEPENDENCIES
## State dependencies

- Transitional / Unstable State when failure is becoming clearer than continuation

## Surface dependencies
- Return-to-Value / Responsive Surface often supportive
- Structurally Valid but Low-Trust Surface may precede stronger confirmation

# STRUCTURAL MEANING
The meaning of this pattern is that the market is not merely pausing after a break attempt; it is structurally failing to hold the attempted directional move and reclaiming prior structure.

# STRUCTURAL INVALIDATION / FAILURE LOGIC
At a high level, the pattern weakens or fails when:
- the reclaim is not actually secured
- renewed acceptance beyond the original break restores continuation authority
- the environment devolves into unstable alternating false moves without clear hierarchy

# MAIN MISREADS / COMPETING INTERPRETATIONS
- ordinary pullback mistaken for failed break
- unresolved breakout mistaken for clear reclaim failure
- unstable chop mistaken for meaningful trap reversal

# PATTERN REJECTION CONDITIONS
Reject or suppress the pattern when:
- rejection is not yet structurally clear
- reclaim of prior structure is incomplete or weak
- continuation logic remains stronger than reversal logic

# FUTURE TESTING / RESEARCH NEEDS

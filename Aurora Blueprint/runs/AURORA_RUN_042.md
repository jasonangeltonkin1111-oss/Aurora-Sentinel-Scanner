The new context contract makes explicit that:
- ASC owns measured world truth
- Aurora consumes that truth and must not fabricate it
- missing ASC surfaces must remain visible as missing truth

### 2. Added a deployability protocol
Aurora now has an explicit law that:
- structure validity is separate from deployability
- spread is a constraint, not an automatic veto
- deployability must compare usable path against execution burden
- horizon may widen inside intraday instead of collapsing immediately to no-trade

### 3. Added a generated strategy-card layer
Aurora now has a clean machine-usable bridge between:
- doctrine files
- wrapper interpretation
- later bounded EA-safe translation

without polluting family or pattern doctrine with false precision.

### 4. Added an anti-starvation ranking / inventory protocol
Aurora now has an explicit way to:
- preserve opportunities
- classify them honestly
- revisit them when context changes

rather than collapsing everything into trade / no-trade.

### 5. Added an EA-safe output boundary
Aurora now has an explicit boundary between:
- machine-safe fields that may later become automatable
- human-only or wrapper-only interpretation that must not be automated prematurely

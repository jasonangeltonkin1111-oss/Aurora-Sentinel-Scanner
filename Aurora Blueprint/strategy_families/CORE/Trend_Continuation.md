# TREND CONTINUATION

## PURPOSE

This file defines the core Aurora Trend Continuation family.

It is written as a family-definition file, not as a final trading ruleset.

Its job is to make the family:
- structurally clear
- habitat-clear
- anti-confusion clear
- easier to translate later into wrapper logic, family cards, and EA logic

---

# 1. FAMILY NAME

## Canonical name
Trend Continuation

## Closely related labels
- directional continuation
- trend family
- TS momentum
- channel break with acceptance

---

# 2. CANONICAL IDEA

Participate when the market is already in a directional discovery / expansion condition and continuation remains stronger than competing balance, trap, or reversal interpretations.

This family is not about guessing the first break.
It is about participating when the directional condition is already structurally established enough that continuation is the better read.

---

# 3. NATIVE HABITAT

Primary native habitats:
- Trend Expansion
- strong Directional Discovery
- clean post-acceptance continuation

This family belongs where:
- the market has already moved beyond balance
- directional performance is real rather than merely attempted
- trust in continuation remains stronger than trust in failure or return-to-balance logic

---

# 4. ANTI-HABITAT

This family should not be treated as native in:
- unstable balance
- unresolved compression
- failed break conditions
- mixed overlap where directional authority is unclear
- exhaustion contexts where continuation trust has materially degraded

---

# 5. STRUCTURAL SIGNATURES

Typical structural signatures include:
- accepted break from prior balance
- directional follow-through
- shallow or orderly retracement behavior
- initiative stronger than responsive opposition
- little evidence that the move is reverting back into accepted rotation

---

# 6. MAIN COMPETING INTERPRETATIONS

Trend Continuation must be distinguished from:
- Breakout / Compression Release
- Failed Break / Trap Reversal
- Balance Rotation / Range Mean-Reversion
- low-trust directional attempt with poor continuation quality

The family is only valid when continuation remains the dominant structural read.

---

# 7. MAIN ANTI-CONFUSIONS

Do not collapse this family with:
- breakout from compression before acceptance
- weak attempted direction
- failed break conditions
- late exhausted movement that only still looks directional geometrically

---

# 8. FUTURE PATTERN DEPENDENCIES

This family will later depend on pattern mappings such as:
- orderly continuation structures
- accepted break and hold patterns
- shallow continuation pullback structures
- post-balance continuation patterns

These pattern children do not need to exist yet for the family to be valid as a family class.

---

# 9. FUTURE WRAPPER / EA TRANSLATION VALUE

A future GPT wrapper or EA should be able to use this file to answer:
- what Trend Continuation is
- where it belongs
- where it does not belong
- what other families it can be confused with

This file should later become one input into a more formal family card or machine-readable family schema.

---

# 10. CURRENT JUDGMENT

Trend Continuation is one of the first core Aurora families because it is a direct child of the current Wave 1 doctrine stack and covers one of the main structural market habitats cleanly.

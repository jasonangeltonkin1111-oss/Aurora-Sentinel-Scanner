# CORE FAMILY INDEX

## PURPOSE

This file is the focused index for the first Aurora core family pack.

It exists so future HQ, GPT wrappers, and EA design layers can enter the core family stack directly without scanning the broader registry first.

---

# 1. FIRST CORE FAMILY PACK

## C-01
- `Trend_Continuation.md`

### Role
Primary continuation family for already-established directional discovery / expansion conditions.

## C-02
- `Trend_Pullback_Continuation.md`

### Role
Continuation family for retracement-within-intact-directional-structure conditions.

## C-03
- `Breakout_Compression_Release.md`

### Role
Transition / release family for compression or balance resolving into meaningful directional expansion.

## C-04
- `Failed_Break_Trap_Reversal.md`

### Role
Failure / reclaim family for directional attempts that lose acceptance and reverse back through initiating structure.

## C-05
- `Balance_Rotation_Range_Mean_Reversion.md`

### Role
Rotational / mean-reversion family for stable balance and range environments.

---

# 2. WHY THIS CORE PACK EXISTS

Together these five families cover the main first-wave structural habitats:
- continuation
- retracement continuation
- release / breakout transition
- failed-state reversal
- balance rotation

That gives Aurora broad structural coverage without requiring immediate dependence on every later data surface.

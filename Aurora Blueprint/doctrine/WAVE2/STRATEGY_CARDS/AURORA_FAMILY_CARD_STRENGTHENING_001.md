This file does not replace the existing family cards.
It identifies what those cards should now hold more strongly because of the Wave 2 extraction.

---

# 1. CARD-LAYER STRENGTHENING RULE

Family cards should not stay as thin restatements of family names.
They should become the main operational compression layer between:
- doctrine
- pattern layer
- future wrapper / EA routing

Wave 2 extraction strengthens that role.

---

# 2. C-01 TREND CONTINUATION — STRENGTHENING

## Add stronger maturity discrimination
The card should distinguish more clearly between:
- fresh continuation
- orderly mature continuation
- late exhausted continuation
- pseudo-continuation inside unstable overlap

## Add stronger decision-zone logic
The card should more explicitly ask whether continuation is occurring:
- through meaningful support/resistance transition
- above a real structural hold
- or only as visual drift without real structural authority

## Add stronger competitor logic
Main competitor families should stay explicit:
- Trend Pullback / Pullback Continuation
- Breakout / Compression Release
- Failed Break / Trap Reversal

---

# 3. C-02 TREND PULLBACK / PULLBACK CONTINUATION — STRENGTHENING

## Add stronger pullback-vs-deterioration distinction
The card should more clearly encode:
- what keeps retracement subordinate
- what shifts it toward deterioration
- what turns it into a likely trap or failed continuation setup

## Add stronger zone-awareness
Pullback quality should be interpreted relative to:
- support/resistance interaction
- prior structural hold zones
- whether the retracement is occurring in a meaningful trend-hosting area

## Add stronger rejection conditions
Reject the family sooner when:
- opposing acceptance strengthens materially
- balance re-formation is becoming dominant
- retracement no longer looks subordinate to the prior move

---

# 4. C-03 BREAKOUT / COMPRESSION RELEASE — STRENGTHENING

## Add stronger release-vs-excursion distinction
The card should continue to sharpen:
- real release vs weak excursion
- accepted break vs temporary escape
- fresh release vs already-mature expansion

## Add stronger failure-path visibility
The card should explicitly preserve how the family can fail into:
- Failed Break / Trap Reversal
- unstable overlap
- weak hold with low-trust continuation

## Add stronger location logic
Breakout quality should remain linked to whether the release is occurring:
- from actual compression or balance
- through meaningful boundary structure
- with real persistence beyond the break

---

# 5. C-04 FAILED BREAK / TRAP REVERSAL — STRENGTHENING

## Add heavier first-class family weight
This family should remain treated as a true primary family, not merely a failed version of a successful breakout.

## Add stronger reclaim logic
The card should make reclaim quality more central:
- reclaim of initiating structure
- clarity of failure hierarchy
- distinction between meaningful reclaim and ordinary noise

## Add stronger competitor boundaries
Competing interpretations that should stay explicit:
- unresolved breakout continuation
- ordinary pullback
- unstable chop masquerading as reclaim reversal

---

# 6. C-05 BALANCE ROTATION / RANGE MEAN-REVERSION — STRENGTHENING

## Add stronger stable-range vs unstable-overlap logic
The card should continue to separate:
- stable accepted rotation
- unstable overlap masquerading as balance
- early breakout pressure emerging inside the range

## Add stronger boundary meaning
Decision-zone and boundary rejection logic should be made more central.

## Add stronger competitor boundaries
Competing interpretations should remain explicit:
- Breakout / Compression Release
- Failed Break / Trap Reversal
- emerging directional migration

---

# 7. CARD CONSEQUENCE FOR FUTURE WRAPPER / EA LAYER

Because of Wave 2 extraction, family cards should increasingly become able to answer:
- what the family is
- what stage of family maturity is present
- what structural zone matters most
- what family is the main competitor
- what should immediately suppress the family

This makes the card layer more useful for routing and ranking later.

---

# 8. DEFERRED BUT PRESERVED RESIDUE

## Later card enrichment residue
- richer candlestick-specific local pattern nuance
- narrower subfamily splits where later justified
- richer performance/evidence annotations
- later quant translation fields for machine-side rule formalization

These remain later enrichments and should not bloat the current cards prematurely.

---

# 9. CURRENT JUDGMENT

Wave 2 extraction now justifies a more mature family-card layer.

The biggest gains are:
- better maturity discrimination
- better decision-zone awareness
- better competitor-family clarity
- stronger rejection conditions

These are the highest-value practical extractions from the current Wave 2 chart books for the card layer.

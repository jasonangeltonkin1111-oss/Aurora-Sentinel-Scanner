This file does not replace the existing pattern cards.
It identifies what those cards should now hold more strongly because of the Wave 2 extraction.

---

# 1. PATTERN-CARD STRENGTHENING RULE

Pattern cards should not remain only broad labels under the atlas.
They should become the main operational compression layer between:
- family cards
- local structural meaning
- failure logic
- future wrapper / EA translation

Wave 2 extraction strengthens that role.

---

# 2. G1 ACCEPTED BREAK AND HOLD — STRENGTHENING

## Add stronger hold-vs-excursion distinction
The card should more clearly encode the difference between:
- genuine structural hold beyond the break
- weak temporary escape
- late exhausted continuation that only still looks like hold

## Add stronger decision-zone meaning
The card should make it more explicit whether the break is occurring:
- through a meaningful support/resistance boundary
- from a real compression or range edge
- with enough persistence to justify continuation interpretation

## Add stronger failure path
The card should preserve how it degrades into:
- Failed Break / Reclaim
- low-trust weak hold
- unstable overlap after apparent release

---

# 3. G2 ORDERLY CONTINUATION PULLBACK — STRENGTHENING

## Add stronger pullback-vs-deterioration distinction
The card should more explicitly separate:
- orderly retracement
- structural deterioration
- early trap or reversal conditions

## Add stronger trend-hosting logic
The card should keep the prior trend context more explicit, so the pattern is not read in isolation from the directional structure that hosts it.

## Add stronger zone-awareness
The card should better reflect whether the retracement is interacting with meaningful structural zones rather than drifting inside noise.

---

# 4. G3 COMPRESSION RELEASE — STRENGTHENING

## Add stronger compression-quality discrimination
The card should more clearly separate:
- meaningful containment before release
- pseudo-compression inside unstable overlap
- already-mature continuation mistaken for fresh release

## Add stronger release-vs-fake-release logic
The card should better preserve the difference between:
- expanding release
- weak excursion
- immediate trap/failure path

## Add stronger family attachment
The card should continue to sit mainly under Breakout / Compression Release, while keeping competitor-family logic visible.

---

# 5. G4 FAILED BREAK / RECLAIM — STRENGTHENING

## Add stronger reclaim hierarchy logic
The card should more clearly encode:
- what makes reclaim meaningful
- what makes it incomplete or weak
- when the pattern is still unresolved rather than truly failed

## Add stronger anti-confusions
Main anti-confusions should remain explicit:
- ordinary pullback
- unresolved breakout
- unstable chop masquerading as trap reversal

## Add stronger decision-zone meaning
Support/resistance and initiating-structure meaning should be even more central to this card.

---

# 6. G5 RANGE REJECTION / RETURN TO VALUE — STRENGTHENING

## Add stronger stable-range vs unstable-overlap distinction
The card should continue to sharpen:
- clean rejection at a meaningful boundary
- unstable overlap that only looks range-like
- early breakout pressure emerging inside the range

## Add stronger rejection-vs-breakout tension
The card should make it more explicit that a boundary reaction is not automatically rejection; it may be early breakout pressure or failure of rejection.

## Add stronger boundary meaning
The card should emphasize that the pattern matters mostly because of where it occurs, not because of a generic local reversal shape.

---

# 7. CARD CONSEQUENCE FOR FUTURE WRAPPER / EA LAYER

Because of Wave 2 extraction, pattern cards should increasingly become able to answer:
- what local structural meaning is present
- what family this pattern belongs to
- what the main failure path is
- what local misread is most dangerous
- what structural zone gives the pattern its meaning

This makes the pattern-card layer more useful for routing and local confirmation later.

---

# 8. DEFERRED BUT PRESERVED RESIDUE

## Later pattern-card enrichment residue
- narrower candlestick-specific child patterns
- richer Bulkowski-style taxonomy splits where evidence justifies them
- session-conditioned pattern branches
- later quant translation fields for measurable pattern testing

These remain later enrichments and should not bloat the current pattern cards prematurely.

---

# 9. CURRENT JUDGMENT

Wave 2 extraction now justifies a more mature pattern-card layer.

The biggest gains are:
- stronger failure-path visibility
- stronger decision-zone meaning
- stronger anti-confusions
- stronger family attachment

These are the highest-value practical extractions from the current Wave 2 chart books for the pattern-card layer.

# 2. HOSTILE-ENVIRONMENT STRENGTHENING ADDITIONS

## 2.1 Credit-contraction hostility rule

### Strengthening statement
Hostile environments can be driven by credit contraction and debt-service stress that deteriorate liquidity, follow-through, and payoff quality.

### Why this is justified
Dalio explicitly frames debt crises as reversals of credit-driven expansions once debt servicing outgrows income capacity. fileciteturn83file0

### Canonical implication
Wave 4 should preserve credit-flow contraction as a distinct hostility class rather than reducing all hostility to volatility or execution friction.

---

## 2.2 Debt-cycle phase rule

### Strengthening statement
Hostility should be distinguished by debt-cycle phase, because bubble-top fragility, depression liquidity stress, and deleveraging recovery are not the same environment.

### Why this is justified
The book explicitly lays out the sequence from early cycle to bubble, top, depression, deleveraging, and normalization. fileciteturn83file0

### Canonical implication
Wave 4 should preserve phase-sensitive hostile-environment mapping whenever macro context is load-bearing.

---

## 2.3 Deflationary-versus-inflationary crisis rule

### Strengthening statement
Macro hostile environments differ materially when the system is facing deflationary debt compression versus inflationary/currency-stress debt compression.

### Why this is justified
Dalio distinguishes the two types explicitly and links the inflationary type more strongly to foreign-currency debt and weaker policy flexibility. fileciteturn83file0

### Canonical implication
Wave 4 should preserve this distinction for later macro-overlay, cross-asset, and FX-aware context work.

---

## 2.4 Policy-regime hostility rule

### Strengthening statement
The same debt stress can produce different market hostility depending on which policy-response levers dominate.

### Why this is justified
Dalio identifies four key policy levers: austerity, restructurings/defaults, money creation/purchases, and transfers. fileciteturn83file0

### Canonical implication
Wave 4 should preserve policy-regime context as part of hostile-environment mapping rather than treating all debt-stress environments as identical.

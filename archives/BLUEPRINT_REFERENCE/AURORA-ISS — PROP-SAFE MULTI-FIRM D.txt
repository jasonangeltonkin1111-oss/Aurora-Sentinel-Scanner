08 — TIER 0: FILE INGESTION + PARSE + VALIDATION (COMPUTE INTEGRITY)
================================================================================

Accepted firm snapshots:
- .csv, .txt

Firm name:
- firm name = filename sans extension, normalize spaces/underscores.

Preferred headers (case/spacing tolerant):
Symbol, Bid, Ask, High, Low, Daily Change, Bid High, Bid Low, Ask High, Ask Low, Open Price, Close Price, Time
Optional extra headers (if present; used deterministically):
InstrumentType, AssetClass, Category, Exchange, SessionHint

Decoding order:
- Attempt encodings in UTF_DECODE_ORDER.

Delimiter detection:
- Try DELIMITER_PRIORITY, accept trailing delimiter and extra empty columns.

Schema lock:
- Required STRING: Symbol
- Required NUMERIC: Bid, Ask, High, Low, BidHigh, BidLow, AskHigh, AskLow
- Optional NUMERIC: Open Price, Close Price, Daily Change
- Optional STRING: Time (telemetry only)
- Symbol is never numeric-coerced.

Numeric coercion:
- Trim whitespace.
- Reject NaN/Inf.
- Percent: strip '%' then parse float.
- Thousands: remove commas then parse.
- Semicolon decimal: if delimiter ';' and value comma-decimal with no dot -> replace ',' with '.' then parse.
- Empty string => NULL (optional only); required numeric cannot be NULL.
- If required numeric fails => row invalid.

Row cleaning:
- Preserve firm-local DISPLAY symbol exactly (do not rewrite beyond trimming).
- Duplicate symbol rows per firm: choose last valid row; log DUPLICATE_ROW_RESOLVED.

Spread compute:
- mid = (Bid + Ask)/2
- spread_bps = ((Ask - Bid)/mid) * BPS_MULTIPLIER

Envelope tolerance:
- tol_px = mid * (ENVELOPE_TOL_PCT / 100)
- Require: High + tol_px >= max(BidHigh, AskHigh)
- Require: Low  - tol_px <= min(BidLow,  AskLow)
No repairs allowed.

Row validity drop if any:
- Symbol blank
- Bid<=0 or Ask<=0 or Ask<=Bid or mid<=0
- High/Low missing or High < Low
- Any of BidHigh/BidLow/AskHigh/AskLow missing
- BidHigh < Bid OR BidLow > Bid
- AskHigh < Ask OR AskLow > Ask
- Envelope violated

Outlier spread handling:
- If spread_bps > SPREAD_BPS_HARD_MAX => OUTLIER_SPREAD (exclude from medians; preserve CORE existence)

All-outlier CORE rule:
- If a CORE has 0 acceptable (non-outlier) rows across all firms => INELIGIBLE for spine ranking.

Tier-0 gate (COMPUTED_TRUE):
- TRUE only if at least 1 firm parsed OK and at least 1 valid row exists.
- If FALSE:
  - run_truth=INVALID
  - validation_warning=TRUE
  - failure_codes += DATA_BROKEN
  - UI prints symbol-only lists + Td/Mkt where possible; numeric fields “—”
  - snapshot still exported FULL-STATE with NA numeric fields.

09 — SYMBOL NORMALIZATION (DISPLAY vs CORE)
================================================================================

- Keep firm-local DISPLAY symbol exactly as in file for UI surfaces.
- Compute CORE for aggregation, medians, and correlation pooling.
- Never merge firm universes.

CORE build (deterministic):
1) S0 = uppercase(trim(DISPLAY))
2) Remove trailing "CASH" or "CASH-<digits>" if present
3) Strip known suffix tokens at end (repeat until no change): [".X", ".OQ", ".C", ".M"]
4) If '.' exists -> take substring before first '.'
5) If '-' exists -> take substring before first '-'
6) FX override: if first 6 chars are currency codes -> CORE=first 6 letters
7) Crypto USDT normalization: if endswith "USDT" -> CORE=first 6 alnum chars
8) Else: CORE = leading alnum block before first separator among [., _, -, space] if length>=3 else CORE=S0
Log CORE_MAP_AMBIGUOUS when it looks FX-like but CORE==DISPLAY.

11 — TIER 1: GLOBAL SPINE (KING — DO NOT ALTER ORDERING)
================================================================================

Median rule:
- odd N: middle; even N: average of two middle (MEDIAN_EVEN_RULE).

Per CORE (acceptable rows only):
- spr_bps_med = median(spread_bps)
- best_firm = argmin spread_bps (tie alphabetical)
- worst_firm = argmax spread_bps (tie reverse alphabetical)
- spread_disp_bps = worst - best
- #F and Acc computed from acceptable firms

Dispersion flag:
- DISPERSION_FLAG TRUE if spread_disp_bps > (DISPERSION_MULT_TH * max(spr_bps_med, EPSILON_REL))

Structure medians:
- mid_med = median(mid)
- range_pct = median(((High-Low)/mid)*100)

Body:
- body_pct = median((abs(Close-Open)/mid)*100 where both exist; else 0)

Wick & effective range:
- wick_pct = max(range_pct - body_pct, 0)
- wick_ratio = wick_pct / max(range_pct, EPSILON_PCT)
- effective_range_pct = max(0.7*range_pct + 0.3*body_pct, EPSILON_PCT)

Move regime (Mv):
- CP if range_pct < 0.30
- NR if 0.30 <= range_pct <= 1.20
- IM if range_pct > 1.20

Execution regime (Ex):
- directional_clarity = clamp(body_pct / max(effective_range_pct, EPSILON_PCT), 0, 1)
- stop_hostility = (1 - directional_clarity) * effective_range_pct
- ST if directional_clarity >= 0.50 AND stop_hostility <= 0.45
- TR if (0.35 <= directional_clarity < 0.55) OR (0.35 < stop_hostility <= 0.75)
- HM otherwise

Gap telemetry:
- daily_change_abs = median(abs(DAILY_CHANGE_PCT)) else UNAVAILABLE
- GapReg QUIET/NORMAL/ELEV/EXT/OUTLIER; if missing => UNDEFINED
- STALLED_GAP TRUE if daily_change_abs > STALLED_GAP_TH AND range_pct < STALLED_RANGE_TH
- GAP_EXHAUST TRUE if daily_change_abs > EXHAUST_GAP_TH AND Ex == HM

Microstructure telemetry:
- spr_bps_peak = ((AskHigh - BidLow)/mid)*BPS_MULTIPLIER
- spr_bps_peak_med = median(spr_bps_peak)
- spr_expansion_bps = max(0, spr_bps_peak_med - spr_bps_med)
- body_friction = spr_bps_med / max(body_pct, EPSILON_PCT)
- impulse_efficiency_score = (body_pct / max(range_pct, EPSILON_PCT)) * range_pct
- MICRO_SPIKE TRUE if spr_bps_peak_med > (MICRO_SPIKE_MULT_TH * max(spr_bps_med, EPSILON_REL))

Adjusted friction (AdjF):
- relative_friction = spr_bps_med / max(effective_range_pct, EPSILON_PCT)
- vol_mod_raw = (1 - STRUCTURE_REWARD_W*directional_clarity) * (1 + CHAOS_PENALTY_W*wick_ratio)
- volatility_modifier = clamp(vol_mod_raw, VOL_MOD_MIN, VOL_MOD_MAX)
- adjusted_friction = relative_friction * volatility_modifier

Pathology flags:
- spread_pct = spr_bps_med / 100.0
- pathology TRUE if effective_range_pct < (spread_pct * K_PATHOLOGY)
- outlier_range TRUE if range_pct > RANGE_PCT_HARD_MAX

PF + selection_score (tie-break only):
- base_penalty_add = 0.25*pathology + 0.25*outlier_range + 0.08*STALLED_GAP + 0.12*GAP_EXHAUST + 0.07*DISPERSION_FLAG + 0.10*MICRO_SPIKE
- PF = 1 + min(base_penalty_add, MAX_TOTAL_PENALTY_ADD)
- selection_score = adjusted_friction * PF

GLOBAL SPINE ordering (IMMUTABLE):
1) adjusted_friction asc
2) selection_score asc
3) CORE asc

Unit lock:
- Spr(bps) always spr_bps_med in bps.

Unit sanity (telemetry only):
- If Ast in {FX, IDX, MET, ENG} AND spr_bps_med < 0.50 => UNIT_SUSPECT=TRUE (no selection impact)

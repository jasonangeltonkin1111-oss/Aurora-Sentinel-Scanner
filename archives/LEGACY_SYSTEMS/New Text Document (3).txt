================================================================================
A) CORE INTENT (WHAT THIS SYSTEM IS)
================================================================================

ISS-X is a deterministic market diagnostics + ranking ecosystem that continuously:
  1) Builds a broker-native Universe from Market Watch (no symbol normalization).
  2) Samples friction (spread) continuously and persistently over time.
  3) Produces StageA candidates using friction quality and absolute stale rules.
  4) Produces structural context (H4/H1 + optional micro M15 refinement) ONLY for StageA.
  5) Produces ranking (Battlefield) + bounded execution modifier (EIM) ONLY for StageA.
  6) Produces correlation-governed final selection of N symbols where SelectionMin ≤ N ≤ SelectionMax.
  7) Produces DeepSpec Mode-B truth capture ONLY for final selected symbols (mode-dependent).
  8) Produces Calendar copy + Fundamentals links (best effort) for selected symbols (mode-dependent).
  9) Produces System Health + Environment regime diagnostics.
 10) Produces an hourly MASTER snapshot containing ALL data for ONLY the selected symbols.

This is not a trading system. It never places, modifies, or manages orders/positions.

================================================================================
B) HARD NON-NEGOTIABLES (ABSOLUTE)
================================================================================

B1) NO TRADING
  - No OrderSend, no positions, no modifications, no pending management.
  - No “blocking” logic that prevents trading elsewhere.
  - No trade permission toggles.

B2) PER-ACCOUNT ISOLATION
  - One terminal + one account + one EA ecosystem instance.
  - Universe = THIS terminal’s Market Watch only.
  - Folder naming MUST isolate by broker + server + login.
  - No cross account/server/broker reading/writing.

B3) DETERMINISM
  - No MathRand / random seeds.
  - Stable iteration order (alphabetical symbol ordering).
  - No hash-map iteration dependency for ordering decisions.
  - All ordering-sensitive floats MUST be quantized before comparisons.
  - Tie-breakers MUST be explicit, stable, and complete.

B4) NO INFERENCE (“DATA TRUTH” RULE)
  - If an MT5 field cannot be read => export value=null AND available=false.
  - Never synthesize values from “similar” fields.
  - Never assume tick_value_profit == tick_value_loss, etc.
  - Derived values are allowed ONLY if all required raw inputs are available,
    and the derived value is labeled as derived and shows raw inputs used.

B5) STALE TICK FLOOR NEVER RELAXES
  - 00_System_MaxTickAgeSeconds is absolute for “fresh tick” qualification.
  - Friction sampling inserts ONLY on fresh valid ticks.
  - StageA eligibility requires fresh tick if 02_UniverseSampling_EnablePriceStreamEligibility=true.
  - Selected[] MUST contain no stale symbols.

B6) NEVER ERASE STATE ON NORMAL RECALC
  - Each stage recompute must continue from prior state:
      * Friction ring buffers prune by time, but do not wipe on recompute.
      * Structural caches, battlefield caches, correlation memory, deepspec queues persist.
  - On startup, stages must load existing .bin state first.
  - Only reset stage state if:
      (a) schema mismatch OR (b) corruption/failed integrity check OR
      (c) explicit forced reset input OR (d) unrecoverable incompatibility.

B7) EXPIRY MUST BE REALISTIC AND DETERMINISTIC
  - “Old” means “not reliable anymore”, not “older than last cycle”.
  - Expiry is stage-specific and deterministic.

B8) OUTPUT RETENTION (PHYSICAL FILES) MUST BE ENFORCED
  - Each stage keeps ONLY the most recent 5 JSON files total in its folder:
      latest.json (always kept) + up to 4 stamped JSON files
  - Older stamped files are deleted deterministically.

B9) MASTER CONTAINS TOP-ONLY
  - Master JSON MUST contain only:
      Selected symbols (SelectionMin..SelectionMax) + their full pipeline data.
  - No universe list, no lightspec array for all symbols, no friction per-symbol for all.
  - Master may include aggregate counts (numbers), but never per-nonselected symbol details.

B10) SPLIT-MODE STAGE ISOLATION (ABSOLUTE)
  DEV_SPLIT_MODE forbids upstream mutation:
  - Stage4 MUST NOT request or execute any Stage1 friction rebuild or threshold relaxation.
  - Stage4 may export friction_relax_needed + recommended thresholds only.
  - No stage calls upstream stage code/functions; only reads upstream JSON files.

B11) SAFE EXECUTION KERNEL REQUIRED (ALL EAs)
  Every EA must include:
  - 1-second timer
  - deterministic scheduler gating (next_due_time)
  - retry backoff gating (retry_after_time)
  - time-budget guarded loops
  - never retry in a tight loop on IO failure
  - never run heavy recompute on every timer tick

E) SAFE EXECUTION KERNEL (MANDATORY FOR ALL EAs)
================================================================================

E0) Timer
  - Every EA uses EventSetTimer(1) and a deterministic internal scheduler.

E1) Scheduler gating (prevents death loops)
  Each EA maintains:
    next_due_stage1_sample_time
    next_due_recompute_time (stage-local or engine-local)
    retry_after_time (for upstream read failures)
    io_backoff_seconds (00_System_IOFailureBackoffSeconds)

  OnTimer() performs ONLY:
    - get now_server
    - check due flags
    - call small bounded work slices
    - never perform unlimited loops

E2) Backoff rules (no retry storms)
  If upstream read fails OR JSON invalid OR file locked:
    set retry_after_time = now + 00_System_IOFailureBackoffSeconds
    export status WAITING/INPUT_INVALID and RETURN
  never retry every 1 second without backoff.

E3) Time budget guards (no terminal freeze)
  Any heavy loop MUST check:
    elapsed_ms > 01_Timing_TimeBudgetMs_PerStageCycle (or stage-specific)
  and stop cleanly, persisting continuation pointers.

E4) Incremental processing pointers (no full recompute storms)
  Each stage with large workloads maintains a persistent rr_index pointer:
    Stage1: rr_index for sampling
    Stage2: struct_rr_index
    Stage3: battle_rr_index
    Stage5: refresh_queue index / rr
  Pointers persist in the stage .bin state.

bool BasicJsonLooksValid(const string &content)
{
   int n = StringLen(content);
   if(n < 2) return false;

   int i=0;
   while(i<n)
   {
      ushort ch = StringGetCharacter(content,i);
      if(ch!=' ' && ch!='\n' && ch!='\r' && ch!='\t') break;
      i++;
   }
   if(i>=n) return false;
   if(StringGetCharacter(content,i)!='{') return false;

   int j=n-1;
   while(j>=0)
   {
      ushort ch2 = StringGetCharacter(content,j);
      if(ch2!=' ' && ch2!='\n' && ch2!='\r' && ch2!='\t') break;
      j--;
   }
   if(j<0) return false;
   if(StringGetCharacter(content,j)!='}') return false;

   return true;
}

// Atomic text write: tmp -> flush -> rename (blueprint pattern)
bool AtomicWriteText(const string rel_final, const string content, string &err_out)
{
   err_out = "";

   if(!BasicJsonLooksValid(content))
   {
      err_out = "content failed basic JSON sanity (must start '{' and end '}')";
      return false;
   }

   string rel_tmp = rel_final + ".tmp";

   ResetLastError();
   int h = FileOpen(rel_tmp, FILE_WRITE|FILE_TXT);
   if(h == INVALID_HANDLE)
   {
      err_out = "FileOpen(tmp) failed err=" + IntegerToString(GetLastError());
      return false;
   }

   FileWriteString(h, content);
   FileFlush(h);
   ulong sz = FileSize(h);
   FileClose(h);

   if(sz < 2)
   {
      err_out = "tmp file too small (" + IntegerToString((int)sz) + ")";
      if(FileIsExist(rel_tmp)) FileDelete(rel_tmp);
      return false;
   }

   // Replace: delete final then move tmp -> final (deterministic, no partial JSON)
   if(FileIsExist(rel_final)) FileDelete(rel_final);

   ResetLastError();
   if(!FileMove(rel_tmp, 0, rel_final, 0))
   {
      err_out = "FileMove(tmp->final) failed err=" + IntegerToString(GetLastError());
      if(FileIsExist(rel_tmp)) FileDelete(rel_tmp);
      return false;
   }

   // Verify final readable
   ResetLastError();
   int vr = FileOpen(rel_final, FILE_READ|FILE_TXT);
   if(vr == INVALID_HANDLE)
   {
      err_out = "Verify open(final) failed err=" + IntegerToString(GetLastError());
      return false;
   }
   FileClose(vr);

   return true;
}

// Keep only samples within FrictionWindowMinutes
void PruneRing(int idx, int now_sec)
{
   int window_sec = FrictionWindowMinutes * 60;
   if(window_sec <= 0) return;

   int cap = g_ring_capacity;
   int cnt = g_fb[idx].count;

   if(cnt <= 0) return;
   if(cnt > cap) cnt = cap;

   int new_t[];
   double new_s[];
   ArrayResize(new_t, cap);
   ArrayResize(new_s, cap);

   int keep = 0;

   for(int j=0;j<cnt;j++)
   {
      int pos = g_fb[idx].head - cnt + j;
      while(pos < 0) pos += cap;
      if(pos >= cap) pos %= cap;

      int ts = g_fb[idx].tsec[pos];
      double sp = g_fb[idx].spread[pos];

      if(ts > 0)
      {
         int age = now_sec - ts;
         if(age >= 0 && age <= window_sec)
         {
            new_t[keep] = ts;
            new_s[keep] = sp;
            keep++;
         }
      }
   }

   for(int k=0;k<cap;k++)
   {
      if(k < keep)
      {
         g_fb[idx].tsec[k] = new_t[k];
         g_fb[idx].spread[k] = new_s[k];
      }
      else
      {
         g_fb[idx].tsec[k] = 0;
         g_fb[idx].spread[k] = 0.0;
      }
   }

   g_fb[idx].count = keep;
   g_fb[idx].head = (keep % cap);
}

            int pos = start_idx + k;
            if(pos >= old_cap) pos -= old_cap;
            if(k < skip) continue;

            g_fb[i].tsec[out] = old_fb[old_idx].tsec[pos];
            g_fb[i].spread[out] = old_fb[old_idx].spread[pos];
            out++;
            if(out >= keep) break;
         }

         g_fb[i].count = keep;
         g_fb[i].head  = (keep % g_ring_capacity);

         // copy last tick probe state (export-only continuity)
         g_tp[i] = old_tp[old_idx];
      }
   }

   if(g_rr_index < 0 || g_rr_index >= g_universe_n) g_rr_index = 0;
}

//==============================
// TICK PROBE (all symbols; does NOT add to friction ring)
//==============================
void ProbeTicksAll(datetime now_server, bool used_fallback_clock)
{
// exported separately in metadata/state

   for(int i=0;i<g_universe_n;i++)
   {
      TickProbe tp;
      tp.tick_available = false;
      tp.bid = 0.0;
      tp.ask = 0.0;
      tp.last = 0.0;
      tp.tick_time = 0;
      tp.tick_age_seconds = INF_AGE;
      tp.tick_age_available = false;

      tp.spread_points = 0.0; tp.spread_points_available = false;
      tp.mid = 0.0; tp.mid_available = false;
      tp.spread_bps = 0.0; tp.spread_bps_available = false;

      tp.stale_flag = true;
      tp.last_error = 0;

      MqlTick tk;
      ResetLastError();
      bool ok = SymbolInfoTick(g_syms[i], tk);
      int err = GetLastError();

      if(ok && !TickLooksValid(tk))
         ok = false;

      if(!ok)
      {
         tp.tick_available = false;
         tp.last_error = err;
         tp.tick_age_seconds = INF_AGE;
         tp.tick_age_available = false;
         tp.stale_flag = true;
         g_tp[i] = tp;
         continue;
      }

      long age = INF_AGE;
      bool age_ok = ComputeTickAge(now_server, tk, age);

      if(!age_ok)
      {
         tp.tick_available = false;
         tp.last_error = 0;
         tp.tick_age_seconds = INF_AGE;
         tp.tick_age_available = false;
         tp.stale_flag = true;
         g_tp[i] = tp;
         continue;
      }

      tp.tick_available = true;
      tp.last_error = 0;
      tp.bid = tk.bid;
      tp.ask = tk.ask;
      tp.last = tk.last;
      tp.tick_time = tk.time;
      tp.tick_age_seconds = age;
      tp.tick_age_available = true;

      // compute spread_points (requires point)
      LightSpec ls = g_ls[i];
      if(ls.point_available && ls.point > 0.0)
      {
         tp.spread_points = (tp.ask - tp.bid) / ls.point;
         tp.spread_points_available = (tp.spread_points >= 0.0);
      }

      // mid + spread_bps
      tp.mid = (tp.bid + tp.ask) * 0.5;
      tp.mid_available = (tp.mid > TINY);
      if(tp.mid_available)
      {
         tp.spread_bps = ((tp.ask - tp.bid) / tp.mid) * 10000.0;
         tp.spread_bps_available = true;
      }

      tp.stale_flag = (!tp.tick_available || tp.tick_age_seconds > (long)MaxTickAgeSeconds);

      g_tp[i] = tp;
   }
}

//==============================
// SAMPLING (Timer A) — ONLY place that adds friction samples
// Insert rule (blueprint):
//   only insert if tick_available=true AND tick_age_seconds <= MaxTickAgeSeconds
//==============================
void SamplingTick()
{
   if(g_universe_n <= 0) return;
   if(MaxSymbolsPerSampleTick <= 0) return;

   bool used_fallback = false;
   datetime now_server = NowServer(used_fallback);
   int now_sec = (int)now_server;

   int budget = MaxSymbolsPerSampleTick;
   if(budget > g_universe_n) budget = g_universe_n;

   for(int j=0;j<budget;j++)
   {
      int idx = g_rr_index;
      g_rr_index++;
      if(g_rr_index >= g_universe_n) g_rr_index = 0;

      // prune within window before adding
      PruneRing(idx, now_sec);

      // tick
      MqlTick tk;
      ResetLastError();
      bool ok = SymbolInfoTick(g_syms[idx], tk);
      int err = GetLastError();

      if(ok && !TickLooksValid(tk))
         ok = false;

      if(!ok)
      {
         // update tick probe cache minimally (sampling does not export directly; recompute probes all)
         g_tp[idx].tick_available = false;
         g_tp[idx].last_error = err;
         g_tp[idx].tick_age_seconds = INF_AGE;
         g_tp[idx].tick_age_available = false;
         g_tp[idx].stale_flag = true;
         continue;
      }

      long age = INF_AGE;
      if(!ComputeTickAge(now_server, tk, age))
      {
         g_tp[idx].tick_available = false;
         g_tp[idx].last_error = 0;
         g_tp[idx].tick_age_seconds = INF_AGE;
         g_tp[idx].tick_age_available = false;
         g_tp[idx].stale_flag = true;
         continue;
      }

      // freshness gate (absolute)
      if(age > (long)MaxTickAgeSeconds)
      {
         g_tp[idx].tick_available = true;
         g_tp[idx].last_error = 0;
         g_tp[idx].bid = tk.bid;
         g_tp[idx].ask = tk.ask;
         g_tp[idx].tick_time = tk.time;
         g_tp[idx].tick_age_seconds = age;
         g_tp[idx].tick_age_available = true;
         g_tp[idx].stale_flag = true;
         continue;
      }

      // must have point for spread_points sample
      if(!g_ls[idx].point_available) continue;
      double pt = g_ls[idx].point;
      if(pt <= 0.0) continue;

      double spread_points = (tk.ask - tk.bid) / pt;
      if(!(spread_points >= 0.0)) continue;

      RingAddSample(idx, now_sec, spread_points);

      // update cached tick probe fields
      g_tp[idx].tick_available = true;
      g_tp[idx].last_error = 0;
      g_tp[idx].bid = tk.bid;
      g_tp[idx].ask = tk.ask;
      g_tp[idx].tick_time = tk.time;
      g_tp[idx].tick_age_seconds = age;
      g_tp[idx].tick_age_available = true;
      g_tp[idx].stale_flag = false;
   }
}

//==============================
// STAGE A METRICS
//==============================
double Mean(const double &x[], int n)
{
   if(n <= 0) return 0.0;
   double sum = 0.0;
   for(int i=0;i<n;i++) sum += x[i];
   return sum / (double)n;
}

double StdDevSample(const double &x[], int n, double mean)
{
   if(n < 2) return 0.0;
   double acc = 0.0;
   for(int i=0;i<n;i++)
   {
      double d = x[i] - mean;
      acc += d*d;
   }
   double var = acc / (double)(n - 1);
   if(var < 0.0) var = 0.0;
   return MathSqrt(var);
}

// deterministic spike threshold relative to mean/std (3-sigma)
int CountSpikes3Sigma(const double &x[], int n, double mean, double sd)
{
   if(n <= 0) return 0;
   if(sd <= 0.0) return 0;
   double thr = mean + 3.0 * sd;
   int c=0;
   for(int i=0;i<n;i++)
      if(x[i] > thr) c++;
   return c;
}

bool GetATRPoints(const string sym, double point, double &atr_points_out)
{
   atr_points_out = 0.0;
   if(point <= 0.0) return false;

   int hATR = iATR(sym, ATR_TF, ATR_Period);
   if(hATR == INVALID_HANDLE) return false;

   double buf[];
   ArrayResize(buf, 1);

   int got = CopyBuffer(hATR, 0, 0, 1, buf);
   IndicatorRelease(hATR);

   if(got < 1) return false;

   double atr_price = buf[0];
   if(atr_price == EMPTY_VALUE) return false;
   if(!(atr_price > 0.0) || atr_price != atr_price) return false;

   double atr_points = atr_price / point;
   if(!(atr_points > 0.0)) return false;

   atr_points_out = atr_points;
   return true;
}

// StageA sort keys (deterministic):
//  1) spread_to_ATR asc (ATR missing sort after)
//  2) spread_bps asc (missing after)
//  3) spread_stability_ratio asc
//  4) spikes asc
//  5) tick_density_proxy desc
//  6) symbol name asc
void SortStageA(StageAItem &arr[], int n)
{
   for(int i=1;i<n;i++)
   {
      StageAItem key = arr[i];
      int j = i - 1;

      while(j >= 0)
      {
         int cmp = 0;

         // 1) spread_to_ATR asc with missing last
         bool key_atr_missing = !key.spread_to_ATR_available;
         bool j_atr_missing   = !arr[j].spread_to_ATR_available;

         if(key_atr_missing != j_atr_missing)
            cmp = (key_atr_missing ? 1 : -1);
         else if(!key_atr_missing)
         {
            if(key.spread_to_ATR < arr[j].spread_to_ATR) cmp = -1;
            else if(key.spread_to_ATR > arr[j].spread_to_ATR) cmp =  1;
         }

         // 2) spread_bps asc with missing last
         if(cmp == 0)
         {
            bool key_bps_missing = !key.spread_bps_available;
            bool j_bps_missing   = !arr[j].spread_bps_available;

            if(key_bps_missing != j_bps_missing)
               cmp = (key_bps_missing ? 1 : -1);
            else if(!key_bps_missing)
            {
               if(key.spread_bps < arr[j].spread_bps) cmp = -1;
               else if(key.spread_bps > arr[j].spread_bps) cmp =  1;
            }
         }

         // 3) stability ratio asc
         if(cmp == 0)
         {
            if(key.spread_stability_ratio < arr[j].spread_stability_ratio) cmp = -1;
            else if(key.spread_stability_ratio > arr[j].spread_stability_ratio) cmp =  1;
         }

         // 4) spikes asc
         if(cmp == 0)
         {
            if(key.spikes < arr[j].spikes) cmp = -1;
            else if(key.spikes > arr[j].spikes) cmp =  1;
         }

         // 5) tick_density_proxy desc
         if(cmp == 0)
         {
            if(key.tick_density_proxy > arr[j].tick_density_proxy) cmp = -1;
            else if(key.tick_density_proxy < arr[j].tick_density_proxy) cmp =  1;
         }

         // 6) symbol asc
         if(cmp == 0)
         {
            if(key.symbol < arr[j].symbol) cmp = -1;
            else if(key.symbol > arr[j].symbol) cmp =  1;
         }

         if(cmp < 0)
         {
            arr[j+1] = arr[j];
            j--;
         }
         else break;
      }

      arr[j+1] = key;
   }
}

// Build StageA list from friction buffers + latest tick probe.
// Candidate eligibility (blueprint):
//  - stale=false (tick_available AND tick_age<=MaxTickAgeSeconds) absolute
//  - samples_count >= MinFrictionSamplesRequired() absolute (for Stage A)
bool BuildStageA(StageAItem &out_stageA[],
                 int &out_n,
                 int &out_stale_symbols_count,
                 int &out_sampled_symbols_count,
                 int &out_fresh_symbols_count,
                 int &out_eligible_symbols_count)
{
   out_n = 0;
   out_stale_symbols_count = 0;
   out_sampled_symbols_count = 0;
   out_fresh_symbols_count = 0;
   out_eligible_symbols_count = 0;

   if(g_universe_n <= 0) return false;

   int Kcap = StageA_K_Default;
   if(Kcap > g_universe_n) Kcap = g_universe_n;
   if(Kcap < 0) Kcap = 0;

   bool used_fallback = false;
   datetime now_server = NowServer(used_fallback);
   int now_sec = (int)now_server;

   int expected = SamplesExpectedCount();
   int min_req  = MinFrictionSamplesRequired();

   StageAItem tmp[];
   ArrayResize(tmp, g_universe_n);
   int cand = 0;

   for(int i=0;i<g_universe_n;i++)
   {
      PruneRing(i, now_sec);

      int cnt = g_fb[i].count;
      if(cnt > 0) out_sampled_symbols_count++;

      TickProbe tp = g_tp[i];
      bool stale = (tp.stale_flag || !tp.tick_available || !tp.tick_age_available || tp.tick_age_seconds > (long)MaxTickAgeSeconds);

      if(stale) out_stale_symbols_count++;
      else out_fresh_symbols_count++;

      // HARD: stale never eligible
      if(stale) continue;

      // HARD: enough samples required
      if(cnt < min_req) continue;

      out_eligible_symbols_count++;

      double xs[];
      ArrayResize(xs, cnt);

      int cap = g_ring_capacity;
      for(int j=0;j<cnt;j++)
      {
         int pos = g_fb[i].head - cnt + j;
         while(pos < 0) pos += cap;
         if(pos >= cap) pos %= cap;
         xs[j] = g_fb[i].spread[pos];
      }

      double mean = Mean(xs, cnt);
      double sd   = StdDevSample(xs, cnt, mean);
      double stab = (mean > TINY ? (sd / mean) : 0.0);
      int spikes  = CountSpikes3Sigma(xs, cnt, mean, sd);

      // spread_bps from tick probe (no inference; if mid missing => unavailable)
      double spread_bps = 0.0;
      bool spread_bps_av = tp.spread_bps_available;

      if(spread_bps_av) spread_bps = tp.spread_bps;

      // ATR points and spread_to_ATR
      double atr_points = 0.0;
      bool atr_av = false;
      bool spread_to_atr_av = false;
      double spread_to_atr = 0.0;

      if(g_ls[i].point_available && g_ls[i].point > 0.0)
      {
         atr_av = GetATRPoints(g_syms[i], g_ls[i].point, atr_points);
      }

      if(atr_av && atr_points > 0.0)
      {
         spread_to_atr = (mean / atr_points);
         spread_to_atr_av = true;
      }
      else
      {
         spread_to_atr = 0.0;
         spread_to_atr_av = false;
      }

      // tick_density_proxy = number of valid ticks sampled per window (proxy)
      int tick_density_proxy = cnt;

      // active_samples_ratio = valid / expected
      double active_ratio = 0.0;
      if(expected > 0) active_ratio = (double)cnt / (double)expected;

      // Quantize ordering-sensitive floats (and export quantized if used for ordering)
      double q_spread_to_atr = spread_to_atr_av ? Q(spread_to_atr) : 0.0;
      double q_spread_bps    = spread_bps_av ? Q(spread_bps) : 0.0;
      double q_stab          = Q(stab);
      double q_active        = Q(active_ratio);

      StageAItem it;
      it.symbol = g_syms[i];
      it.universe_index = i;

      it.samples_count = cnt;
      it.samples_expected_count = expected;
      it.active_samples_ratio = q_active;

      it.spread_mean_60m = Q(mean);
      it.spread_std_60m  = Q(sd);
      it.spread_stability_ratio = q_stab;

      it.spread_bps = q_spread_bps;
      it.spread_bps_available = spread_bps_av;

      it.atr_points = atr_points;
      it.atr_points_available = atr_av;

      it.spread_to_ATR = q_spread_to_atr;
      it.spread_to_ATR_available = spread_to_atr_av;

      it.spikes = spikes;
      it.tick_density_proxy = tick_density_proxy;

      it.tick_available = tp.tick_available;
      it.tick_age_seconds = tp.tick_age_seconds;

      it.stageA_rank = 0;

      tmp[cand] = it;
      cand++;
   }

   ArrayResize(tmp, cand);
   SortStageA(tmp, cand);

   int take = Kcap;
   if(take > cand) take = cand;

   ArrayResize(out_stageA, take);
   for(int i=0;i<take;i++)
   {
      out_stageA[i] = tmp[i];
      out_stageA[i].stageA_rank = i + 1;
   }

   out_n = take;
   return true;
}

//==============================
// BINARY STATE PERSISTENCE (friction ring buffers + rr_index + universe fp)
// Buffer continuity rule: do not wipe unless schema mismatch/corruption/universe remap.
//==============================
string StateFileRel(){ return (g_out_folder + "iss_friction_state.bin"); }

// simple FNV-1a checksum over ring content (export-only)
ulong ComputeFrictionChecksum()
{
   ulong h = 1469598103934665603ULL;

   // include universe fp
   string ufp = g_universe_fp;
   for(int k=0;k<StringLen(ufp);k++)
      h = Fnv1a64_Update(h, StringGetCharacter(ufp,k));

   // include rr_index and ring content
   string rr = IntegerToString(g_rr_index);
   for(int k=0;k<StringLen(rr);k++)
      h = Fnv1a64_Update(h, StringGetCharacter(rr,k));

   for(int i=0;i<g_universe_n;i++)
   {
      string s = g_syms[i];
      for(int k=0;k<StringLen(s);k++)
         h = Fnv1a64_Update(h, StringGetCharacter(s,k));
      h = Fnv1a64_Update(h, (ushort)'|');

      int cnt = g_fb[i].count;
      if(cnt > g_ring_capacity) cnt = g_ring_capacity;
      if(cnt < 0) cnt = 0;

      for(int j=0;j<cnt;j++)
      {
         int pos = g_fb[i].head - cnt + j;
         while(pos < 0) pos += g_ring_capacity;
         if(pos >= g_ring_capacity) pos %= g_ring_capacity;

         int ts = g_fb[i].tsec[pos];
         // fold ts bytes
         h = Fnv1a64_Update(h, (ushort)(ts & 0xFF));
         h = Fnv1a64_Update(h, (ushort)((ts >> 8) & 0xFF));

         // fold quantized spread
         long spq = (long)MathRound(Q(g_fb[i].spread[pos]) * 1000000.0);
         h = Fnv1a64_Update(h, (ushort)(spq & 0xFF));
         h = Fnv1a64_Update(h, (ushort)((spq >> 8) & 0xFF));
      }
   }

   return h;
}

bool SaveFrictionState(string &err_out)
{
   err_out = "";
   string fn  = StateFileRel();
   string tmp = fn + ".tmp";

   ResetLastError();
   int h = FileOpen(tmp, FILE_WRITE|FILE_BIN);
   if(h == INVALID_HANDLE)
   {
      err_out = "Save: FileOpen(tmp) failed err=" + IntegerToString(GetLastError());
      return false;
   }

   // header
   FileWriteString(h, ISS_VERSION);
   FileWriteString(h, SCHEMA_VERSION);

   // config fields (export-only; state continuity does not reset on param tweaks; schema gates validity)
   FileWriteInteger(h, SampleSeconds, INT_VALUE);
   FileWriteInteger(h, FrictionWindowMinutes, INT_VALUE);
   FileWriteInteger(h, MaxTickAgeSeconds, INT_VALUE);
   FileWriteInteger(h, RecomputeMinutes, INT_VALUE);
   FileWriteInteger(h, MaxSymbolsPerSampleTick, INT_VALUE);

   // universe identity
   FileWriteInteger(h, g_universe_n, INT_VALUE);
   FileWriteInteger(h, g_ring_capacity, INT_VALUE);
   FileWriteString(h, g_universe_fp);

   // write per symbol
   for(int i=0;i<g_universe_n;i++)
   {
      FileWriteString(h, g_syms[i]);

      // ring meta
      FileWriteInteger(h, g_fb[i].count, INT_VALUE);
      FileWriteInteger(h, g_fb[i].head, INT_VALUE);

      // ring arrays (fixed cap)
      for(int k=0;k<g_ring_capacity;k++)
         FileWriteInteger(h, g_fb[i].tsec[k], INT_VALUE);
      for(int k=0;k<g_ring_capacity;k++)
         FileWriteDouble(h, g_fb[i].spread[k]);
   }

   // rr pointer
   FileWriteInteger(h, g_rr_index, INT_VALUE);

   // checksum
   ulong cks = ComputeFrictionChecksum();
   FileWriteLong(h, (long)cks);

   FileFlush(h);
   FileClose(h);

   if(!AtomicReplaceBinary(fn, tmp, err_out))
   {
      err_out = "Save: " + err_out;
      if(FileIsExist(tmp)) FileDelete(tmp);
      return false;
   }

   return true;
}

bool LoadFrictionState(string &err_out)
{
   err_out = "";
   string fn = StateFileRel();

   if(!FileIsExist(fn))
   {
      err_out = "state file missing (first run)";
      return false;
   }

   ResetLastError();
   int h = FileOpen(fn, FILE_READ|FILE_BIN);
   if(h == INVALID_HANDLE)
   {
      err_out = "Load: FileOpen failed err=" + IntegerToString(GetLastError());
      return false;
   }

   string ver = FileReadString(h);
   string schema = FileReadString(h);

   // read config fields (not used to reset state; schema gates validity)
   int ss = FileReadInteger(h, INT_VALUE);
   int fwm = FileReadInteger(h, INT_VALUE);
   int mta = FileReadInteger(h, INT_VALUE);
   int rm = FileReadInteger(h, INT_VALUE);
   int ms = FileReadInteger(h, INT_VALUE);

   int n_file = FileReadInteger(h, INT_VALUE);
   int cap_file = FileReadInteger(h, INT_VALUE);
   string fp_file = FileReadString(h);

   if(schema != SCHEMA_VERSION)
   {
      FileClose(h);
      err_out = "Load: schema mismatch";
      return false;
   }

   if(n_file < 0) n_file = 0;
   if(cap_file < 1) cap_file = 1;
   if(cap_file > MAX_RING_HARD_CAP) cap_file = MAX_RING_HARD_CAP;

   // read per symbol data into current universe by remap
   for(int j=0;j<n_file;j++)
   {
      if(FileIsEnding(h)) break;

      string sym = FileReadString(h);
      int cnt = FileReadInteger(h, INT_VALUE);
      int head = FileReadInteger(h, INT_VALUE);

      if(cnt < 0) cnt = 0;
      if(cnt > cap_file) cnt = cap_file;
      if(head < 0 || head >= cap_file) head = 0;

      int tarr[];
      double sarr[];
      ArrayResize(tarr, cap_file);
      ArrayResize(sarr, cap_file);

      for(int k=0;k<cap_file;k++)
      {
         if(FileIsEnding(h)) { tarr[k]=0; break; }
         tarr[k] = FileReadInteger(h, INT_VALUE);
      }
      for(int k=0;k<cap_file;k++)
      {
         if(FileIsEnding(h)) { sarr[k]=0.0; break; }
         sarr[k] = FileReadDouble(h);
      }

      // locate current index
      int idx = -1;
      for(int i=0;i<g_universe_n;i++)
      {
         if(g_syms[i] == sym) { idx = i; break; }
      }
      if(idx < 0) continue;

      // copy newest samples into current ring cap
      int keep = cnt;
      if(keep > g_ring_capacity) keep = g_ring_capacity;

      int start_idx = head - cnt;
      while(start_idx < 0) start_idx += cap_file;
      if(start_idx >= cap_file) start_idx %= cap_file;

      int skip = cnt - keep;
      int out = 0;
      for(int k=0;k<cnt;k++)
      {
         int pos = start_idx + k;
         if(pos >= cap_file) pos -= cap_file;
         if(k < skip) continue;

         g_fb[idx].tsec[out] = tarr[pos];
         g_fb[idx].spread[out] = sarr[pos];
         out++;
         if(out >= keep) break;
      }

      g_fb[idx].count = keep;
      g_fb[idx].head  = (keep % g_ring_capacity);
   }

   // rr_index
   if(!FileIsEnding(h))
   {
      int rr = FileReadInteger(h, INT_VALUE);
      if(rr >= 0 && rr < g_universe_n) g_rr_index = rr;
   }

   // checksum (best effort)
   long cks_file = 0;
   if(!FileIsEnding(h))
      cks_file = FileReadLong(h);

   FileClose(h);

   // verify checksum (best effort; if mismatch => treat as read error to preserve determinism)
   ulong cks_now = ComputeFrictionChecksum();
   if(cks_file != 0 && (ulong)cks_file != cks_now)
   {
      err_out = "Load: checksum mismatch";
      return false;
   }

   return true;
}

   final_error = "";

// Keep only samples within FrictionWindowMinutes
void PruneRing(int idx, int now_sec)
{
   int window_sec = FrictionWindowMinutes * 60;
   if(window_sec <= 0) return;

   int cap = g_ring_capacity;
   int cnt = g_fb[idx].count;

   if(cnt <= 0) return;
   if(cnt > cap) cnt = cap;

   int new_t[];
   double new_s[];
   ArrayResize(new_t, cap);
   ArrayResize(new_s, cap);

   int keep = 0;

   for(int j=0;j<cnt;j++)
   {
      int pos = g_fb[idx].head - cnt + j;
      while(pos < 0) pos += cap;
      if(pos >= cap) pos %= cap;

      int ts = g_fb[idx].tsec[pos];
      double sp = g_fb[idx].spread[pos];

      if(ts > 0)
      {
         int age = now_sec - ts;
         if(age >= 0 && age <= window_sec)
         {
            new_t[keep] = ts;
            new_s[keep] = sp;
            keep++;
         }
      }
   }

   for(int k=0;k<cap;k++)
   {
      if(k < keep)
      {
         g_fb[idx].tsec[k] = new_t[k];
         g_fb[idx].spread[k] = new_s[k];
      }
      else
      {
         g_fb[idx].tsec[k] = 0;
         g_fb[idx].spread[k] = 0.0;
      }
   }

   g_fb[idx].count = keep;
   g_fb[idx].head = (keep % cap);
}

   int cap = g_ring_capacity;

   g_fb[idx].tsec[g_fb[idx].head] = tsec;
   g_fb[idx].spread[g_fb[idx].head] = spread_points;

   g_fb[idx].head++;
   if(g_fb[idx].head >= cap) g_fb[idx].head = 0;

   if(g_fb[idx].count < cap) g_fb[idx].count++;
}

//==============================
// MIN FRICTION SAMPLES REQUIRED (deterministic, blueprint-derived)
// MinFrictionSamplesRequired = deterministic minimum derived from window/sample.
// Here: exact expected samples in window.
//==============================
int SamplesExpectedCount()
{
   int window_sec = FrictionWindowMinutes * 60;
   if(SampleSeconds <= 0 || window_sec <= 0) return 0;
   return (window_sec / SampleSeconds);
}
int MinFrictionSamplesRequired()
{
   int exp = SamplesExpectedCount();
   if(exp < 1) exp = 1;
   return exp;
}

//==============================
// BUILD UNIVERSE (Market Watch)
// - Universe ordering stabilized deterministically (alphabetical raw symbol)
// - Must not wipe friction buffers on normal recompute
// - On universe change: remap by symbol string, preserve existing buffers
//==============================
void BuildUniverseAndRemapFriction(const bool first_run)
{
   // build new universe list (sorted)
   int total = SymbolsTotal(true);
   if(total < 0) total = 0;

   string new_syms[];
   ArrayResize(new_syms, total);

   int count = 0;
   for(int i=0;i<total;i++)
   {
      string s = SymbolName(i,true);
      if(s=="") continue;
      SymbolSelect(s,true);
      new_syms[count] = s;
      count++;
   }

   ArrayResize(new_syms, count);
   ArraySort(new_syms);

   // compute new fingerprint
   string new_fp = ComputeUniverseFingerprintFromList(new_syms);

   // ring capacity based on current params
   int exp = SamplesExpectedCount();
   int new_cap = exp;
   if(new_cap < 16) new_cap = 16;
   if(new_cap > MAX_RING_HARD_CAP) new_cap = MAX_RING_HARD_CAP;

   // allocate new arrays
   LightSpec new_ls[];
   TickProbe new_tp[];
   FrictionBuf new_fb[];
   // --- Capture OLD state before overwrite ---
int old_n = g_universe_n;
int old_cap = g_ring_capacity;

string old_syms[];
ArrayResize(old_syms, old_n);

LightSpec old_ls[];
ArrayResize(old_ls, old_n);

TickProbe old_tp[];
ArrayResize(old_tp, old_n);

FrictionBuf old_fb[];
ArrayResize(old_fb, old_n);

for(int i=0;i<old_n;i++)
{
   old_syms[i] = g_syms[i];
   old_ls[i]   = g_ls[i];
   old_tp[i]   = g_tp[i];

   old_fb[i].head  = g_fb[i].head;
   old_fb[i].count = g_fb[i].count;

   ArrayResize(old_fb[i].tsec, ArraySize(g_fb[i].tsec));
   ArrayResize(old_fb[i].spread, ArraySize(g_fb[i].spread));

   for(int k=0;k<ArraySize(g_fb[i].tsec);k++)
   {
      old_fb[i].tsec[k]   = g_fb[i].tsec[k];
      old_fb[i].spread[k] = g_fb[i].spread[k];
   }
}
   ArrayResize(new_ls, count);
   ArrayResize(new_tp, count);
   ArrayResize(new_fb, count);

   // if first run or old universe empty, just init
   if(first_run || g_universe_n <= 0)
   {
      ArrayResize(g_syms, count);
for(int i=0;i<count;i++)
   g_syms[i] = new_syms[i];
      g_universe_n = count;
      g_universe_fp = new_fp;

      g_ring_capacity = new_cap;

      ArrayResize(g_ls, g_universe_n);
      ArrayResize(g_tp, g_universe_n);
      ArrayResize(g_fb, g_universe_n);

      for(int i=0;i<g_universe_n;i++)
      {
         g_ls[i] = new_ls[i];
         g_tp[i] = new_tp[i];
         g_fb[i] = new_fb[i];
         RingInit(i);
      }

      if(g_rr_index < 0 || g_rr_index >= g_universe_n) g_rr_index = 0;
      return;
   }



   // adopt new universe
   g_universe_n = count;
   g_universe_fp = new_fp;
   g_ring_capacity = new_cap;

   ArrayResize(g_ls, g_universe_n);
   ArrayResize(g_tp, g_universe_n);
   ArrayResize(g_fb, g_universe_n);

   for(int i=0;i<g_universe_n;i++)
   {
      g_ls[i] = new_ls[i];
      g_tp[i] = new_tp[i];

      // find symbol in old
      int old_idx = -1;
      for(int j=0;j<old_n;j++)
      {
         if(old_syms[j] == g_syms[i]) { old_idx = j; break; }
      }

      // init ring
      RingInit(i);

      // copy ring if exists
      if(old_idx >= 0)
      {
         // copy newest samples (chronological) into new ring cap
         int src_cnt = old_fb[old_idx].count;
         int src_head = old_fb[old_idx].head;
         if(src_cnt > old_cap) src_cnt = old_cap;
         if(src_cnt < 0) src_cnt = 0;

         int keep = src_cnt;
         if(keep > g_ring_capacity) keep = g_ring_capacity;

         // locate start idx in old ring (chronological)
         int start_idx = src_head - src_cnt;
         while(start_idx < 0) start_idx += old_cap;
         if(start_idx >= old_cap) start_idx %= old_cap;

         int skip = src_cnt - keep;
         int out = 0;
         for(int k=0;k<src_cnt;k++)
         {
            int pos = start_idx + k;
            if(pos >= old_cap) pos -= old_cap;
            if(k < skip) continue;

            g_fb[i].tsec[out] = old_fb[old_idx].tsec[pos];
            g_fb[i].spread[out] = old_fb[old_idx].spread[pos];
            out++;
            if(out >= keep) break;
         }

         g_fb[i].count = keep;
         g_fb[i].head  = (keep % g_ring_capacity);

         // copy last tick probe state (export-only continuity)
         g_tp[i] = old_tp[old_idx];
      }
   }

   if(g_rr_index < 0 || g_rr_index >= g_universe_n) g_rr_index = 0;
}

//==============================
// TICK PROBE (all symbols; does NOT add to friction ring)
//==============================
void ProbeTicksAll(datetime now_server, bool used_fallback_clock)
{
// exported separately in metadata/state

   for(int i=0;i<g_universe_n;i++)
   {
      TickProbe tp;
      tp.tick_available = false;
      tp.bid = 0.0;
      tp.ask = 0.0;
      tp.last = 0.0;
      tp.tick_time = 0;
      tp.tick_age_seconds = INF_AGE;
      tp.tick_age_available = false;

      tp.spread_points = 0.0; tp.spread_points_available = false;
      tp.mid = 0.0; tp.mid_available = false;
      tp.spread_bps = 0.0; tp.spread_bps_available = false;

      tp.stale_flag = true;
      tp.last_error = 0;

      MqlTick tk;
      ResetLastError();
      bool ok = SymbolInfoTick(g_syms[i], tk);
      int err = GetLastError();

      if(ok && !TickLooksValid(tk))
         ok = false;

      if(!ok)
      {
         tp.tick_available = false;
         tp.last_error = err;
         tp.tick_age_seconds = INF_AGE;
         tp.tick_age_available = false;
         tp.stale_flag = true;
         g_tp[i] = tp;
         continue;
      }

      long age = INF_AGE;
      bool age_ok = ComputeTickAge(now_server, tk, age);

      if(!age_ok)
      {
         tp.tick_available = false;
         tp.last_error = 0;
         tp.tick_age_seconds = INF_AGE;
         tp.tick_age_available = false;
         tp.stale_flag = true;
         g_tp[i] = tp;
         continue;
      }

      tp.tick_available = true;
      tp.last_error = 0;
      tp.bid = tk.bid;
      tp.ask = tk.ask;
      tp.last = tk.last;
      tp.tick_time = tk.time;
      tp.tick_age_seconds = age;
      tp.tick_age_available = true;

      // compute spread_points (requires point)
      LightSpec ls = g_ls[i];
      if(ls.point_available && ls.point > 0.0)
      {
         tp.spread_points = (tp.ask - tp.bid) / ls.point;
         tp.spread_points_available = (tp.spread_points >= 0.0);
      }

      // mid + spread_bps
      tp.mid = (tp.bid + tp.ask) * 0.5;
      tp.mid_available = (tp.mid > TINY);
      if(tp.mid_available)
      {
         tp.spread_bps = ((tp.ask - tp.bid) / tp.mid) * 10000.0;
         tp.spread_bps_available = true;
      }

      tp.stale_flag = (!tp.tick_available || tp.tick_age_seconds > (long)MaxTickAgeSeconds);

      g_tp[i] = tp;
   }
}

//==============================
// SAMPLING (Timer A) — ONLY place that adds friction samples
// Insert rule (blueprint):
//   only insert if tick_available=true AND tick_age_seconds <= MaxTickAgeSeconds
//==============================
void SamplingTick()
{
   if(g_universe_n <= 0) return;
   if(MaxSymbolsPerSampleTick <= 0) return;

   bool used_fallback = false;
   datetime now_server = NowServer(used_fallback);
   int now_sec = (int)now_server;

   int budget = MaxSymbolsPerSampleTick;
   if(budget > g_universe_n) budget = g_universe_n;

   for(int j=0;j<budget;j++)
   {
      int idx = g_rr_index;
      g_rr_index++;
      if(g_rr_index >= g_universe_n) g_rr_index = 0;

      // prune within window before adding
      PruneRing(idx, now_sec);

      // tick
      MqlTick tk;
      ResetLastError();
      bool ok = SymbolInfoTick(g_syms[idx], tk);
      int err = GetLastError();

      if(ok && !TickLooksValid(tk))
         ok = false;

      if(!ok)
      {
         // update tick probe cache minimally (sampling does not export directly; recompute probes all)
         g_tp[idx].tick_available = false;
         g_tp[idx].last_error = err;
         g_tp[idx].tick_age_seconds = INF_AGE;
         g_tp[idx].tick_age_available = false;
         g_tp[idx].stale_flag = true;
         continue;
      }

      long age = INF_AGE;
      if(!ComputeTickAge(now_server, tk, age))
      {
         g_tp[idx].tick_available = false;
         g_tp[idx].last_error = 0;
         g_tp[idx].tick_age_seconds = INF_AGE;
         g_tp[idx].tick_age_available = false;
         g_tp[idx].stale_flag = true;
         continue;
      }

      // freshness gate (absolute)
      if(age > (long)MaxTickAgeSeconds)
      {
         g_tp[idx].tick_available = true;
         g_tp[idx].last_error = 0;
         g_tp[idx].bid = tk.bid;
         g_tp[idx].ask = tk.ask;
         g_tp[idx].tick_time = tk.time;
         g_tp[idx].tick_age_seconds = age;
         g_tp[idx].tick_age_available = true;
         g_tp[idx].stale_flag = true;
         continue;
      }

      // must have point for spread_points sample
      if(!g_ls[idx].point_available) continue;
      double pt = g_ls[idx].point;
      if(pt <= 0.0) continue;

      double spread_points = (tk.ask - tk.bid) / pt;
      if(!(spread_points >= 0.0)) continue;

      RingAddSample(idx, now_sec, spread_points);

      // update cached tick probe fields
      g_tp[idx].tick_available = true;
      g_tp[idx].last_error = 0;
      g_tp[idx].bid = tk.bid;
      g_tp[idx].ask = tk.ask;
      g_tp[idx].tick_time = tk.time;
      g_tp[idx].tick_age_seconds = age;
      g_tp[idx].tick_age_available = true;
      g_tp[idx].stale_flag = false;
   }
}

//==============================
// STAGE A METRICS
//==============================
double Mean(const double &x[], int n)
{
   if(n <= 0) return 0.0;
   double sum = 0.0;
   for(int i=0;i<n;i++) sum += x[i];
   return sum / (double)n;
}

double StdDevSample(const double &x[], int n, double mean)
{
   if(n < 2) return 0.0;
   double acc = 0.0;
   for(int i=0;i<n;i++)
   {
      double d = x[i] - mean;
      acc += d*d;
   }
   double var = acc / (double)(n - 1);
   if(var < 0.0) var = 0.0;
   return MathSqrt(var);
}

// deterministic spike threshold relative to mean/std (3-sigma)
int CountSpikes3Sigma(const double &x[], int n, double mean, double sd)
{
   if(n <= 0) return 0;
   if(sd <= 0.0) return 0;
   double thr = mean + 3.0 * sd;
   int c=0;
   for(int i=0;i<n;i++)
      if(x[i] > thr) c++;
   return c;
}

bool GetATRPoints(const string sym, double point, double &atr_points_out)
{
   atr_points_out = 0.0;
   if(point <= 0.0) return false;

   int hATR = iATR(sym, ATR_TF, ATR_Period);
   if(hATR == INVALID_HANDLE) return false;

   double buf[];
   ArrayResize(buf, 1);

   int got = CopyBuffer(hATR, 0, 0, 1, buf);
   IndicatorRelease(hATR);

   if(got < 1) return false;

   double atr_price = buf[0];
   if(atr_price == EMPTY_VALUE) return false;
   if(!(atr_price > 0.0) || atr_price != atr_price) return false;

   double atr_points = atr_price / point;
   if(!(atr_points > 0.0)) return false;

   atr_points_out = atr_points;
   return true;
}

// StageA sort keys (deterministic):
//  1) spread_to_ATR asc (ATR missing sort after)
//  2) spread_bps asc (missing after)
//  3) spread_stability_ratio asc
//  4) spikes asc
//  5) tick_density_proxy desc
//  6) symbol name asc
void SortStageA(StageAItem &arr[], int n)
{
   for(int i=1;i<n;i++)
   {
      StageAItem key = arr[i];
      int j = i - 1;

      while(j >= 0)
      {
         int cmp = 0;

         // 1) spread_to_ATR asc with missing last
         bool key_atr_missing = !key.spread_to_ATR_available;
         bool j_atr_missing   = !arr[j].spread_to_ATR_available;

         if(key_atr_missing != j_atr_missing)
            cmp = (key_atr_missing ? 1 : -1);
         else if(!key_atr_missing)
         {
            if(key.spread_to_ATR < arr[j].spread_to_ATR) cmp = -1;
            else if(key.spread_to_ATR > arr[j].spread_to_ATR) cmp =  1;
         }

         // 2) spread_bps asc with missing last
         if(cmp == 0)
         {
            bool key_bps_missing = !key.spread_bps_available;
            bool j_bps_missing   = !arr[j].spread_bps_available;

            if(key_bps_missing != j_bps_missing)
               cmp = (key_bps_missing ? 1 : -1);
            else if(!key_bps_missing)
            {
               if(key.spread_bps < arr[j].spread_bps) cmp = -1;
               else if(key.spread_bps > arr[j].spread_bps) cmp =  1;
            }
         }

         // 3) stability ratio asc
         if(cmp == 0)
         {
            if(key.spread_stability_ratio < arr[j].spread_stability_ratio) cmp = -1;
            else if(key.spread_stability_ratio > arr[j].spread_stability_ratio) cmp =  1;
         }

         // 4) spikes asc
         if(cmp == 0)
         {
            if(key.spikes < arr[j].spikes) cmp = -1;
            else if(key.spikes > arr[j].spikes) cmp =  1;
         }

         // 5) tick_density_proxy desc
         if(cmp == 0)
         {
            if(key.tick_density_proxy > arr[j].tick_density_proxy) cmp = -1;
            else if(key.tick_density_proxy < arr[j].tick_density_proxy) cmp =  1;
         }

         // 6) symbol asc
         if(cmp == 0)
         {
            if(key.symbol < arr[j].symbol) cmp = -1;
            else if(key.symbol > arr[j].symbol) cmp =  1;
         }

         if(cmp < 0)
         {
            arr[j+1] = arr[j];
            j--;
         }
         else break;
      }

      arr[j+1] = key;
   }
}

// Build StageA list from friction buffers + latest tick probe.
// Candidate eligibility (blueprint):
//  - stale=false (tick_available AND tick_age<=MaxTickAgeSeconds) absolute
//  - samples_count >= MinFrictionSamplesRequired() absolute (for Stage A)
bool BuildStageA(StageAItem &out_stageA[],
                 int &out_n,
                 int &out_stale_symbols_count,
                 int &out_sampled_symbols_count,
                 int &out_fresh_symbols_count,
                 int &out_eligible_symbols_count)
{
   out_n = 0;
   out_stale_symbols_count = 0;
   out_sampled_symbols_count = 0;
   out_fresh_symbols_count = 0;
   out_eligible_symbols_count = 0;

   if(g_universe_n <= 0) return false;

   int Kcap = StageA_K_Default;
   if(Kcap > g_universe_n) Kcap = g_universe_n;
   if(Kcap < 0) Kcap = 0;

   bool used_fallback = false;
   datetime now_server = NowServer(used_fallback);
   int now_sec = (int)now_server;

   int expected = SamplesExpectedCount();
   int min_req  = MinFrictionSamplesRequired();

   StageAItem tmp[];
   ArrayResize(tmp, g_universe_n);
   int cand = 0;

   for(int i=0;i<g_universe_n;i++)
   {
      PruneRing(i, now_sec);

      int cnt = g_fb[i].count;
      if(cnt > 0) out_sampled_symbols_count++;

      TickProbe tp = g_tp[i];
      bool stale = (tp.stale_flag || !tp.tick_available || !tp.tick_age_available || tp.tick_age_seconds > (long)MaxTickAgeSeconds);

      if(stale) out_stale_symbols_count++;
      else out_fresh_symbols_count++;

      // HARD: stale never eligible
      if(stale) continue;

      // HARD: enough samples required
      if(cnt < min_req) continue;

      out_eligible_symbols_count++;

      double xs[];
      ArrayResize(xs, cnt);

      int cap = g_ring_capacity;
      for(int j=0;j<cnt;j++)
      {
         int pos = g_fb[i].head - cnt + j;
         while(pos < 0) pos += cap;
         if(pos >= cap) pos %= cap;
         xs[j] = g_fb[i].spread[pos];
      }

      double mean = Mean(xs, cnt);
      double sd   = StdDevSample(xs, cnt, mean);
      double stab = (mean > TINY ? (sd / mean) : 0.0);
      int spikes  = CountSpikes3Sigma(xs, cnt, mean, sd);

      // spread_bps from tick probe (no inference; if mid missing => unavailable)
      double spread_bps = 0.0;
      bool spread_bps_av = tp.spread_bps_available;

      if(spread_bps_av) spread_bps = tp.spread_bps;

      // ATR points and spread_to_ATR
      double atr_points = 0.0;
      bool atr_av = false;
      bool spread_to_atr_av = false;
      double spread_to_atr = 0.0;

      if(g_ls[i].point_available && g_ls[i].point > 0.0)
      {
         atr_av = GetATRPoints(g_syms[i], g_ls[i].point, atr_points);
      }

      if(atr_av && atr_points > 0.0)
      {
         spread_to_atr = (mean / atr_points);
         spread_to_atr_av = true;
      }
      else
      {
         spread_to_atr = 0.0;
         spread_to_atr_av = false;
      }

      // tick_density_proxy = number of valid ticks sampled per window (proxy)
      int tick_density_proxy = cnt;

      // active_samples_ratio = valid / expected
      double active_ratio = 0.0;
      if(expected > 0) active_ratio = (double)cnt / (double)expected;

      // Quantize ordering-sensitive floats (and export quantized if used for ordering)
      double q_spread_to_atr = spread_to_atr_av ? Q(spread_to_atr) : 0.0;
      double q_spread_bps    = spread_bps_av ? Q(spread_bps) : 0.0;
      double q_stab          = Q(stab);
      double q_active        = Q(active_ratio);

      StageAItem it;
      it.symbol = g_syms[i];
      it.universe_index = i;

      it.samples_count = cnt;
      it.samples_expected_count = expected;
      it.active_samples_ratio = q_active;

      it.spread_mean_60m = Q(mean);
      it.spread_std_60m  = Q(sd);
      it.spread_stability_ratio = q_stab;

      it.spread_bps = q_spread_bps;
      it.spread_bps_available = spread_bps_av;

      it.atr_points = atr_points;
      it.atr_points_available = atr_av;

      it.spread_to_ATR = q_spread_to_atr;
      it.spread_to_ATR_available = spread_to_atr_av;

      it.spikes = spikes;
      it.tick_density_proxy = tick_density_proxy;

      it.tick_available = tp.tick_available;
      it.tick_age_seconds = tp.tick_age_seconds;

      it.stageA_rank = 0;

      tmp[cand] = it;
      cand++;
   }

   ArrayResize(tmp, cand);
   SortStageA(tmp, cand);

   int take = Kcap;
   if(take > cand) take = cand;

   ArrayResize(out_stageA, take);
   for(int i=0;i<take;i++)
   {
      out_stageA[i] = tmp[i];
      out_stageA[i].stageA_rank = i + 1;
   }

   out_n = take;
   return true;
}

//==============================
// BINARY STATE PERSISTENCE (friction ring buffers + rr_index + universe fp)
// Buffer continuity rule: do not wipe unless schema mismatch/corruption/universe remap.
//==============================
string StateFileRel(){ return (g_out_folder + "iss_friction_state.bin"); }

// simple FNV-1a checksum over ring content (export-only)
ulong ComputeFrictionChecksum()
{
   ulong h = 1469598103934665603ULL;

   // include universe fp
   string ufp = g_universe_fp;
   for(int k=0;k<StringLen(ufp);k++)
      h = Fnv1a64_Update(h, StringGetCharacter(ufp,k));

   // include rr_index and ring content
   string rr = IntegerToString(g_rr_index);
   for(int k=0;k<StringLen(rr);k++)
      h = Fnv1a64_Update(h, StringGetCharacter(rr,k));

   for(int i=0;i<g_universe_n;i++)
   {
      string s = g_syms[i];
      for(int k=0;k<StringLen(s);k++)
         h = Fnv1a64_Update(h, StringGetCharacter(s,k));
      h = Fnv1a64_Update(h, (ushort)'|');

      int cnt = g_fb[i].count;
      if(cnt > g_ring_capacity) cnt = g_ring_capacity;
      if(cnt < 0) cnt = 0;

      for(int j=0;j<cnt;j++)
      {
         int pos = g_fb[i].head - cnt + j;
         while(pos < 0) pos += g_ring_capacity;
         if(pos >= g_ring_capacity) pos %= g_ring_capacity;

         int ts = g_fb[i].tsec[pos];
         // fold ts bytes
         h = Fnv1a64_Update(h, (ushort)(ts & 0xFF));
         h = Fnv1a64_Update(h, (ushort)((ts >> 8) & 0xFF));

         // fold quantized spread
         long spq = (long)MathRound(Q(g_fb[i].spread[pos]) * 1000000.0);
         h = Fnv1a64_Update(h, (ushort)(spq & 0xFF));
         h = Fnv1a64_Update(h, (ushort)((spq >> 8) & 0xFF));
      }
   }

   return h;
}

bool SaveFrictionState(string &err_out)
{
   err_out = "";
   string fn  = StateFileRel();
   string tmp = fn + ".tmp";

   ResetLastError();
   int h = FileOpen(tmp, FILE_WRITE|FILE_BIN);
   if(h == INVALID_HANDLE)
   {
      err_out = "Save: FileOpen(tmp) failed err=" + IntegerToString(GetLastError());
      return false;
   }

   // header
   FileWriteString(h, ISS_VERSION);
   FileWriteString(h, SCHEMA_VERSION);

   // config fields (export-only; state continuity does not reset on param tweaks; schema gates validity)
   FileWriteInteger(h, SampleSeconds, INT_VALUE);
   FileWriteInteger(h, FrictionWindowMinutes, INT_VALUE);
   FileWriteInteger(h, MaxTickAgeSeconds, INT_VALUE);
   FileWriteInteger(h, RecomputeMinutes, INT_VALUE);
   FileWriteInteger(h, MaxSymbolsPerSampleTick, INT_VALUE);

   // universe identity
   FileWriteInteger(h, g_universe_n, INT_VALUE);
   FileWriteInteger(h, g_ring_capacity, INT_VALUE);
   FileWriteString(h, g_universe_fp);

   // write per symbol
   for(int i=0;i<g_universe_n;i++)
   {
      FileWriteString(h, g_syms[i]);

      // ring meta
      FileWriteInteger(h, g_fb[i].count, INT_VALUE);
      FileWriteInteger(h, g_fb[i].head, INT_VALUE);

      // ring arrays (fixed cap)
      for(int k=0;k<g_ring_capacity;k++)
         FileWriteInteger(h, g_fb[i].tsec[k], INT_VALUE);
      for(int k=0;k<g_ring_capacity;k++)
         FileWriteDouble(h, g_fb[i].spread[k]);
   }

   // rr pointer
   FileWriteInteger(h, g_rr_index, INT_VALUE);

   // checksum
   ulong cks = ComputeFrictionChecksum();
   FileWriteLong(h, (long)cks);

   FileFlush(h);
   FileClose(h);

   if(!AtomicReplaceBinary(fn, tmp, err_out))
   {
      err_out = "Save: " + err_out;
      if(FileIsExist(tmp)) FileDelete(tmp);
      return false;
   }

   return true;
}

bool LoadFrictionState(string &err_out)
{
   err_out = "";
   string fn = StateFileRel();

   if(!FileIsExist(fn))
   {
      err_out = "state file missing (first run)";
      return false;
   }

   ResetLastError();
   int h = FileOpen(fn, FILE_READ|FILE_BIN);
   if(h == INVALID_HANDLE)
   {
      err_out = "Load: FileOpen failed err=" + IntegerToString(GetLastError());
      return false;
   }

   string ver = FileReadString(h);
   string schema = FileReadString(h);

   // read config fields (not used to reset state; schema gates validity)
   int ss = FileReadInteger(h, INT_VALUE);
   int fwm = FileReadInteger(h, INT_VALUE);
   int mta = FileReadInteger(h, INT_VALUE);
   int rm = FileReadInteger(h, INT_VALUE);
   int ms = FileReadInteger(h, INT_VALUE);

   int n_file = FileReadInteger(h, INT_VALUE);
   int cap_file = FileReadInteger(h, INT_VALUE);
   string fp_file = FileReadString(h);

   if(schema != SCHEMA_VERSION)
   {
      FileClose(h);
      err_out = "Load: schema mismatch";
      return false;
   }

   if(n_file < 0) n_file = 0;
   if(cap_file < 1) cap_file = 1;
   if(cap_file > MAX_RING_HARD_CAP) cap_file = MAX_RING_HARD_CAP;

   // read per symbol data into current universe by remap
   for(int j=0;j<n_file;j++)
   {
      if(FileIsEnding(h)) break;

      string sym = FileReadString(h);
      int cnt = FileReadInteger(h, INT_VALUE);
      int head = FileReadInteger(h, INT_VALUE);

      if(cnt < 0) cnt = 0;
      if(cnt > cap_file) cnt = cap_file;
      if(head < 0 || head >= cap_file) head = 0;

      int tarr[];
      double sarr[];
      ArrayResize(tarr, cap_file);
      ArrayResize(sarr, cap_file);

      for(int k=0;k<cap_file;k++)
      {
         if(FileIsEnding(h)) { tarr[k]=0; break; }
         tarr[k] = FileReadInteger(h, INT_VALUE);
      }
      for(int k=0;k<cap_file;k++)
      {
         if(FileIsEnding(h)) { sarr[k]=0.0; break; }
         sarr[k] = FileReadDouble(h);
      }

      // locate current index
      int idx = -1;
      for(int i=0;i<g_universe_n;i++)
      {
         if(g_syms[i] == sym) { idx = i; break; }
      }
      if(idx < 0) continue;

      // copy newest samples into current ring cap
      int keep = cnt;
      if(keep > g_ring_capacity) keep = g_ring_capacity;

      int start_idx = head - cnt;
      while(start_idx < 0) start_idx += cap_file;
      if(start_idx >= cap_file) start_idx %= cap_file;

      int skip = cnt - keep;
      int out = 0;
      for(int k=0;k<cnt;k++)
      {
         int pos = start_idx + k;
         if(pos >= cap_file) pos -= cap_file;
         if(k < skip) continue;

         g_fb[idx].tsec[out] = tarr[pos];
         g_fb[idx].spread[out] = sarr[pos];
         out++;
         if(out >= keep) break;
      }

      g_fb[idx].count = keep;
      g_fb[idx].head  = (keep % g_ring_capacity);
   }

   // rr_index
   if(!FileIsEnding(h))
   {
      int rr = FileReadInteger(h, INT_VALUE);
      if(rr >= 0 && rr < g_universe_n) g_rr_index = rr;
   }

   // checksum (best effort)
   long cks_file = 0;
   if(!FileIsEnding(h))
      cks_file = FileReadLong(h);

   FileClose(h);

   // verify checksum (best effort; if mismatch => treat as read error to preserve determinism)
   ulong cks_now = ComputeFrictionChecksum();
   if(cks_file != 0 && (ulong)cks_file != cks_now)
   {
      err_out = "Load: checksum mismatch";
      return false;
   }

   return true;
}

//==============================
// EXPORT WRAPPER
//==============================
bool ExportSnapshot(StageAItem &stageA[],
                    int stageA_n,
                    bool used_fallback_clock,
                    bool universe_change_detected,
                    string universe_change_reason,
                    int stale_symbols_count,
                    int sampled_symbols_count,
                    int fresh_symbols_count,
                    int eligible_symbols_count,
                    string &final_error)
{
   final_error = "";

   datetime t_server = NowServer(used_fallback_clock);
   string stamp = Stamp(t_server);

   string snapshot = BuildSnapshotJson(stageA, stageA_n, used_fallback_clock,
                                       universe_change_detected, universe_change_reason,
                                       stale_symbols_count, sampled_symbols_count,
                                       fresh_symbols_count, eligible_symbols_count);

   string err;

   if(AtomicWriteEnabled)
   {
      if(!AtomicWriteText(g_out_folder+"iss_snapshot_latest.json", snapshot, err))
      {
         final_error = "snapshot_latest: " + err;
         return false;
      }
      if(KeepRollingSnapshots)
      {
         if(!AtomicWriteText(g_out_folder+"iss_snapshot_"+stamp+".json", snapshot, err))
         {
            final_error = "snapshot_stamp: " + err;
            return false;
         }
      }
   }
   else
   {
      // non-atomic (discouraged; kept only because AtomicWriteEnabled is a parameter)
      ResetLastError();
      int h = FileOpen(g_out_folder+"iss_snapshot_latest.json", FILE_WRITE|FILE_TXT);
      if(h == INVALID_HANDLE)
      {
         final_error = "snapshot_latest non-atomic open failed err=" + IntegerToString(GetLastError());
         return false;
      }
      FileWriteString(h, snapshot);
      FileClose(h);

      if(KeepRollingSnapshots)
      {
         h = FileOpen(g_out_folder+"iss_snapshot_"+stamp+".json", FILE_WRITE|FILE_TXT);
         if(h == INVALID_HANDLE)
         {
            final_error = "snapshot_stamp non-atomic open failed err=" + IntegerToString(GetLastError());
            return false;
         }
         FileWriteString(h, snapshot);
         FileClose(h);
      }
   }

   if(!SaveFrictionState(err))
   {
      final_error = "save_state: " + err;
      return false;
   }

   return true;
}

//==============================
// RECOMPUTE CYCLE (Timer B)
// Step order preserved:
//  0 Metadata snapshot (within BuildSnapshotJson)
//  1 Universe discovery
//  2 LightSpec + TickProbe
//  3 Friction sampling is Timer A (separate); here we only prune + compute
//  4 StageA build + export
//==============================
void RecomputeCycle()
{
   // Detect universe changes by fingerprint (not count only)
   string fp_now="";
   int mw_count=0;
   ComputeMarketWatchUniverseFingerprint(fp_now, mw_count);

   bool universe_changed = (mw_count != g_universe_n || fp_now != g_universe_fp);
   string universe_change_reason = "NO_CHANGE";
   if(g_universe_n == 0 && mw_count > 0) universe_change_reason = "FIRST_RUN";
   else if(universe_changed) universe_change_reason = "MARKET_WATCH_CHANGED";

   if(universe_changed || g_universe_n == 0)
   {
      BuildUniverseAndRemapFriction(g_universe_n == 0);

      // load persisted state after universe build/remap
      string load_err = "";
      bool loaded_ok = LoadFrictionState(load_err);
      if(loaded_ok)
      {
         g_state_load_status = "OK";
         g_state_load_message = "";
      }
      else
      {
         if(load_err == "state file missing (first run)")
            g_state_load_status = "MISSING";
         else
            g_state_load_status = "READ_ERROR";

         g_state_load_message = load_err;
         if(load_err != "")
            Print("ISSX_DEV_Friction: state not loaded after universe rebuild: ", load_err);
      }
   }

   // LightSpec scan (all universe symbols)
   CaptureLightSpecAll();

   // Tick probe scan (all universe symbols) — does not add to ring
   bool used_fallback = false;
   datetime now_server = NowServer(used_fallback);
   ProbeTicksAll(now_server, used_fallback);

   // prune rings to current window
   int now_sec = (int)now_server;
   for(int i=0;i<g_universe_n;i++)
      PruneRing(i, now_sec);

   // build StageA
   StageAItem stageA[];
   int stageA_n = 0;

   int stale_symbols_count = 0;
   int sampled_symbols_count = 0;
   int fresh_symbols_count = 0;
   int eligible_symbols_count = 0;

   BuildStageA(stageA, stageA_n,
               stale_symbols_count, sampled_symbols_count,
               fresh_symbols_count, eligible_symbols_count);

   // export
   string err;
   bool ok = ExportSnapshot(stageA, stageA_n,
                            used_fallback,
                            universe_changed,
                            universe_change_reason,
                            stale_symbols_count,
                            sampled_symbols_count,
                            fresh_symbols_count,
                            eligible_symbols_count,
                            err);

   if(ok)
   {
      Print("ISSX_DEV_Friction: export OK. Universe=", g_universe_n,
            " StageA=", stageA_n,
            " Stale=", stale_symbols_count,
            " SampledSyms=", sampled_symbols_count,
            " Eligible=", eligible_symbols_count,
            " Folder=MQL5\\Files\\", g_out_folder);
   }
   else
   {
      Print("ISSX_DEV_Friction: EXPORT ERROR: ", err);
   }
}

int OnInit()
{
   BuildDevFolder();

   // build universe and rings
   BuildUniverseAndRemapFriction(true);

   g_state_load_status = "NOT_ATTEMPTED";
   g_state_load_message = "";

   string err;
   bool ok = LoadFrictionState(err);
   if(ok)
   {
      g_state_load_status = "OK";
      g_state_load_message = "";
   }
   else
   {
      if(err == "state file missing (first run)")
         g_state_load_status = "MISSING";
      else
         g_state_load_status = "READ_ERROR";

      g_state_load_message = err;
      if(err != "")
         Print("ISSX_DEV_Friction: state not loaded: ", err);
   }

   // initial scans
   CaptureLightSpecAll();

   bool used_fallback = false;
   datetime now_server = NowServer(used_fallback);
   ProbeTicksAll(now_server, used_fallback);

   g_last_sample_time = now_server;
   g_last_recompute_time = now_server;

   // 1-second timer; internal cadence enforces SampleSeconds and RecomputeMinutes deterministically
   EventSetTimer(1);

   Print("ISSX_DEV_Friction: START. Folder=MQL5\\Files\\", g_out_folder,
         " Universe=", g_universe_n,
         " Fingerprint=", g_universe_fp,
         " RingCap=", g_ring_capacity,
         " SampleSeconds=", SampleSeconds,
         " WindowMin=", FrictionWindowMinutes,
         " RecomputeMin=", RecomputeMinutes,
         " MinFrictionSamplesRequired=", MinFrictionSamplesRequired(),
         " StateLoad=", g_state_load_status);

   return(INIT_SUCCEEDED);
}

void OnDeinit(const int reason)
{
   EventKillTimer();

   string err;
   SaveFrictionState(err);

   Print("ISSX_DEV_Friction: STOP. reason=", reason);
}

void OnTimer()
{
   bool used_fallback = false;
   datetime now_server = NowServer(used_fallback);

   // Timer A sampling cadence
   if(SampleSeconds > 0 && (now_server - g_last_sample_time) >= SampleSeconds)
   {
      g_last_sample_time = now_server;
      SamplingTick();
   }

   // Timer B recompute cadence
   if(RecomputeMinutes > 0 && (now_server - g_last_recompute_time) >= (RecomputeMinutes * 60))
   {
      g_last_recompute_time = now_server;
      RecomputeCycle();
   }
}

string IssJsonEscape(string s)
{
   StringReplace(s, "\\", "\\\\");
   StringReplace(s, "\"", "\\\"");
   StringReplace(s, "\r", "\\r");
   StringReplace(s, "\n", "\\n");
   StringReplace(s, "\t", "\\t");
   return s;
}
string IssB2S(bool v){ return (v ? "true" : "false"); }
string IssI2S(int v){ return IntegerToString(v); }
string IssL2S(long v){ return StringFormat("%I64d", v); }
string IssD2S(double v, int digits){ return DoubleToString(v, digits); }

double IssQuantize(double v, int decimals)
{
   return NormalizeDouble(v, decimals);
}

bool IssBasicJsonLooksValid(const string &content)
{
   int n = StringLen(content);
   if(n < 2) return false;

   int i=0;
   while(i<n)
   {
      ushort ch = StringGetCharacter(content,i);
      if(ch!=' ' && ch!='\n' && ch!='\r' && ch!='\t') break;
      i++;
   }
   if(i>=n) return false;
   if(StringGetCharacter(content,i)!='{') return false;

   int j=n-1;
   while(j>=0)
   {
      ushort ch2 = StringGetCharacter(content,j);
      if(ch2!=' ' && ch2!='\n' && ch2!='\r' && ch2!='\t') break;
      j--;
   }
   if(j<0) return false;
   if(StringGetCharacter(content,j)!='}') return false;
   return true;
}

{
   MqlDateTime st; TimeToStruct(t_server, st);
   return StringFormat("%04d%02d%02d_%02d%02d%02d", st.year, st.mon, st.day, st.hour, st.min, st.sec);
}

bool IssEnsureFolder(string folder_rel)
{
   ResetLastError();
   if(FolderCreate(folder_rel)) return true;
   int err = GetLastError();
   if(err == 5019 || err == 0) return true;
   return false;
}

datetime IssNowServer(bool &used_fallback)
{
   used_fallback = false;
   datetime t = TimeTradeServer();
   if(t <= 0)

      t = TimeCurrent();

   return t;
}

//==================================================================
// UTIL: FNV-1a 64-bit over UTF-16 code units (ushort)
//==================================================================
ulong IssFnv1a64_Update(ulong h, ushort ch)
{
   h ^= (ulong)ch;
   h *= FNV64_PRIME;
   return h;
}

ulong IssFnv1a64_UpdateStr(ulong h, const string &s)
{
   int len = StringLen(s);
   for(int i=0;i<len;i++)
      h = IssFnv1a64_Update(h, StringGetCharacter(s,i));
   return h;
}

string IssFnv1a64_HashToString(ulong h)
{
   return StringFormat("%I64d",(long)h);
}

//==================================================================
// UTIL: Fingerprint of ordered universe (broker-native symbols, alphabetical)
//==================================================================
string IssComputeUniverseFingerprintFromList(string &syms_sorted[])
{
   ulong h = FNV64_OFFSET_BASIS;
   int n = ArraySize(syms_sorted);
   for(int i=0;i<n;i++)
   {
      h = IssFnv1a64_UpdateStr(h, syms_sorted[i]);
      h = IssFnv1a64_Update(h, (ushort)'|');
   }
   return IssFnv1a64_HashToString(h);
}

//==================================================================
// UTIL: Canonical params fingerprinting (FULL + STAGE SCOPE)
// Order MUST be fixed and stable.
//==================================================================
ulong IssHashAddKV(ulong h, const string &k, const string &v)
{
   h = IssFnv1a64_UpdateStr(h, k);
   h = IssFnv1a64_Update(h, (ushort)'=');
   h = IssFnv1a64_UpdateStr(h, v);
   h = IssFnv1a64_Update(h, (ushort)';');
   return h;
}

# Aurora Archive

This folder is the preserved Aurora research library and conceptual archive.

It contains books, research references, and major blueprint-style text files that may support future Aurora development.

This folder is immutable.
Only additive markdown navigation files may be added.

## What this folder is for
- future research extraction
- concept mapping
- long-horizon design support
- identifying possible future rulecard or model sources

## What this folder is not for
- current Phase 1 scanner expansion
- direct MT5 implementation source
- strategy dumping into product code

## Indexes
- `AURORA_FUTURE_INDEX.md` — future-use conceptual index
- `ASC_PHASE1_INDEX.md` — narrow current-task relevance index

# ASC Publication

This domain owns:
- symbol-file building
- placeholder scaffolds
- temp writes
- validation
- safe promotion to final files
- summary publication
- bridge-ready metadata such as schema version and generated-at timestamps

Publication renders prepared truth.
It does not invent or compute truth.

## Output law

Publication surfaces should stay:
- operator-readable
- machine-parseable
- explicit about Reserved and Pending sections
- safe for future Aurora-side ingestion without leaking strategy logic into scanner files

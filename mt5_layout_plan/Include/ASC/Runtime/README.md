# ASC Runtime

This domain owns:
- heartbeat control
- restore flow
- due scheduling
- dispatch order
- bounded work
- degraded-mode handling
- input/menu wiring for runtime-owned settings
- staged verification hooks for runtime-owned behavior

This domain does not own trader-facing wording.

## Placeholder law

Even before later layers are implemented, the runtime domain should reserve clean extension points for:
- Layer 2 snapshot dispatch
- Layer 3 filter dispatch
- Layer 4 selection dispatch
- Layer 5 deep-analysis dispatch

Those placeholders should remain explicit and honest rather than absent.
